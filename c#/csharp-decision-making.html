C# Decision Making




What is decision making

Think about a situation wherer you want to execute different statements based on a specific condition

Let's say you and your friends are planning to go for a tour, but you didn't decide where to go!. Now you and your want to decide where to go based on the budget.

All your friends has contributed some money. Now is the challenge

If the budget is == $1000 ? you and your friends can visit Goa. else if the budget is $2000 you and your friends can visit New Delhi, else if the budget is more than $3000 you and your friends can visit Mumbai....etc

Think about how does your gmail inbox emails works?
If the user selects display 100 emails per page? then it displays 100 emails per page.
If the user selects display 50 emails per page then display only 50 emails per page.
If the user selects display 200 emails per page then it displays 200 emails per page.

This is how decision making works in programming

C# provides 2 types of conditional statements. Those are if and switch statements. Lets see all possible statements with examples

if statement in c#

It functions mainly based on boolean expression by one or more statements.

Syntax for c# if statement 

if(boolean_expression)
{
        /* statement will execute if the boolean expression is true */
}
Example program for c# if statement 

using System;
namespace DecisionMaking
{
   class IfStatementExample
   {
      static void Main(string[] args)
      {
         /* local variable definition */
         int age = 16;
         /* check the boolean condition using if statement */
         if (age < 18)
         {
            /* if condition is true then print the following */
            Console.WriteLine("You are a minor");
         }
         Console.WriteLine("Your age is just: {0}", age);
         Console.ReadLine();
      }
   }
}

If you run the above code, it will produce the following output

Output
You are a minor
Your age is just: 16

If else statemtn in C#

An if statement can be followed by an optional else statement, which executes when the boolean expression is false.

Syntax for if else statement in c#


    if(boolean_condition)
{
   /* Execute this block if the boolean expression is true */
}
else
{
   /* Execute this block if the boolean expression is false */
}
If else example program in C# 

using System;
namespace DecisionMaking
{
   class ExampleProgram
   {
      static void Main(string[] args)
      {
         /* local variable definition */
         int marks = 34;
         
         /* check the boolean condition */
         if (marks > 35)
         {
            /* if condition is true then print the following */
            Console.WriteLine("You are pass");
         }
         else
         {
            /* if condition is false then print the following */
            Console.WriteLine("Sorry,You are failed");
         }
         Console.WriteLine("Your total marks is {0}",marks);
         Console.ReadLine();
      }
   }
}
Output
Sorry,You are failed
Your total marks is 34

Else if statement in c#

Else if statement will works if all the above if and else conditions are failed.

In some case you want to say if condition A, condition B failed then then do something else. Thats where the else if condition comes

Syntax of else if statement in C#

if(Condition A) 1)
{
   /* Executes when the condition  A is true */
}
else if( Condition B)
{
   /* Executes when the condition B is true */
}
else if( condition C)
{
   /* Executes when the condition C is true */
}
else 
{
   /* executes when the none of the above conditions is true */
}
Example program for else if statement in C#

using System;
namespace DecisionMaking
{
   class ExampleElseIfStatment
   {
      static void Main(string[] args)
      {
         /* local variables definition */
         int total_budget_collected_in_dollars = 2000;
         int budget_required_to_goa_in_dollars = 1000
         int budget_required_to_newdelhi_in_dollars = 2000
         int budget_required_to_mumbai_in_dollars = 1000

         /* check the boolean condition using if statement */
         if (total_budget_collected_in_dollars >=  budget_required_to_goa_in_dollars)
         {
            /* if condition is true then print the following */
            Console.WriteLine("We can visit Goa");
         }
         else if (total_budget_collected_in_dollars >=  budget_required_to_newdelhi_in_dollars )
         {
             Console.WriteLine("We can visit New Delhi");
         }
         else
         {
          Console.WriteLine("We can visit Mumbai");   
         }
         
         Console.ReadLine();
      }
   }
}

When you run the above example it will produce the following output

Output
We can visit New Delhi

Switch Statement in C#

A switch statement is a type of selection control mechanism used to allow the value of a variable or expression to change the control flow of program execution via a multiway branch.

A simple case statement in real life would be something like below

You are tavelling from one place to another.
When you reach Bangelore, it print Bangelore
When you reach New Delhi, it print Bangelore
When you reach Jaipur, it print Bangelore
When you reach Mumabi, it print Bangelore
if not, the default case statement like your source will print.

A typical switch statement would be like this


switch (age) {
  case 1:  Console.WriteLine("You're one.");     break;
  case 2:  Console.WriteLine("You're two.");     break;
  case 3:  Console.WriteLine("You're three.");   break;
  case 4:  Console.WriteLine("You're four.");    break;
  default: Console.WriteLine("You're neither!"); break;
}
syntax of a switch statement in C#: 

switch(expression) {
   case 1  :
      statement(s);
      break; /* optional */
   case  2:
      statement(s);
      break; /* optional */
  
   /* you can have any number of case statements */
   default : /* Optional */
   statement(s);
}
Example program for C# switch statement 

using System;
namespace DecisionMaking
{
   class ExampleSwitchProgram
   {
      static void Main(string[] args)
      {
         /* local variable definition */
          string train_location;
          train_location = "Mumbai";
        
         
         switch (train_location)
         {
            case "Bangelore":
               Console.WriteLine("You are in Bangelore");
    
           
               break;
            case "New Delhi":
               Console.WriteLine("You are in New Delhi");
               break;
            case "Mumbai":
               Console.WriteLine("You are in Mumabi");
               break;
               default:
            Console.WriteLine("You are in Hyderabad");
               break;
         }
         Console.WriteLine("The train is currently moving from  {0}", train_location);
         Console.ReadLine();
      }
   }
}

Output
You are in Mumabi
The train is currently moving from Mumbai

The ?: Operator

In C# we can also perform if else statement decision making with conditional operator ? :

Syntax 

ExpressionX ? ExpressionY : ExpressionZ;

Example: a>b ? “a is bigger” : “b is bigger”;

From the above example, the first expression is evaluated before finalizing the result of the statement. So if a is greater than b then it returns the result as “a is bigger”.
