What is C# ?




What is C#

C# is an object-oriented programming language from Microsoft that aims to combine the computing power of C++ with the programming ease of Visual Basic. C# is based on C++ and contains features similar to those of Java.

C# is an object-oriented programming language developed by Microsoft as part of the.Net initiative & later approved as a standard by ECMA & ISO.

Anders Hejlsberg leads the development of language which has procedural object-oriented syntax based on C++ and includes influences from several other programming languages.

Why C#?

There are many languages exist before ‘c’ language like Cobol, Pascal, and Fortran etc…for developing the applications.

COBOL - Used for developing the Business Applications

PASCAL - Used for developing the Scientific Applications

FORTRAN- Used for developing the Mathematical Applications

The disadvantage of all above languages is “If a common man want to develop Business, Scientific and Mathematical related applications, he has to learn all above languages. Meaning, All applications can not be developed by a single language.

C language:

To avoid this kind of problem “C language” comes into existence in 1960’s. It is also called as a Procedural Programming Language or Generalized language.

But, C language has disadvantages in providing the Code Security and Code Reusability.

Reusability: - Reusability means duplication of code in the program. Due to this issue, the programming size becomes too big.

C++:

Disadvantages of c language overcome by c++(OOP) in 1970’s. It is a successful language.C++ happen to be an extension of ‘c’ that means every feature exist in c present in c++ as well.

C++ is a platform dependent meaning when we download any application, it can’t run on other operating systems. In 1990, ’s this drawback is identified when internet becomes so popular.

C# is an elegant and type-safe object-oriented language that enables developers to build a variety of secure and robust applications that run on the .NET Framework. You can use C# to create Windows client applications, XML Web services, distributed components, client-server applications, database applications, and much, much more. Visual C# provides an futuristic code editor, convenient user interface designers, integrated debugger, and many other tools to make it easier to develop applications based on the C# language and the .NET Framework.

Design goals of C#:

It is intended to be a simple,modern,general purpose and object oriented language. The language should include strong type checking, array bound checking, detection of attempts to use uninitialized variable, source code portability and automatic memory management. The language is intended for use in developing software components that can take advantage of distributed environments.

Programmer portability is very important in software industry, so especially for those programmers already familiar with c,c++ and c# will be the best choice. Support for internationalization. C# is intended to be suitable for writing applications both to hosted and embedded systems.
