C# Classes




What is a Class?

In object-oriented programming, a class is an extensible program-code-template for creating objects, providing initial values for state (member variables) and implementations of behavior (member functions, methods).

In an object-oriented programming language, a program is the collection of members like variables and functions. But those members are enclosed under a special container known as a “class”. Which gives the basic security for contents that are present inside and we call the class as the wrapper.

A class is a user-defined data type very much same as a structure in our procedural programming languages.

As a class is a data type to consume the class or call the members of class we must create an object.

We can not consume a data type directly whether it is predefined or less defined. So, to consume a datatype first we need to create a copy of the type. 

  int = 100;          // invalid
  int x = 100;        //valid

In the above case x is a copy of type ‘int’ and the memory allocation is done for that x only. For the data type, there is no allocation of memory.

Exactly in the same way if we define a class or a structure here also datatypes only but user defined. So, to consume them also we need to create a copy of types.

Copies of simple types like int, float, char etc are known as variables whereas copies of complex types like user defined structures or class will be called as “object”.


Syntax to define a class in c#: 
[<modifiers>] class <name>
{
        --- Define members (variables and methods)
}
Note:

[] means optional , <> means any

Modifiers:

Modifiers are special keywords that can be used on a class like public, static, abstract, sealed etc. Using these modifiers is only optional.

Class:

The class is a keyword to tell we are defining a class. Note: c# is a case-sensitive language. So, all the keywords must be in lowercase and while consuming any predefined libraries we need to refer them in the proper case. Proper case means first character of every word is capital and rest will be in small. 

  WriteLine()
  ReadLine()
Name:

Name represents the name of a class we want to define. Where class names are user-defined and we can follow any case for our class name. but it will be better if we also adopt proper case just like libraries.

The file in which we define the class must be saved with “.cs” extension and the name of the file can be anything, but it will be better if the file name is same as class.


Example program of C# class

using System;

namespace ConsoleApplication
{
    class ExampleClass
    {
        static void Main(string[] args)
        {
            Car car;

    }
