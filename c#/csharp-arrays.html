C# Arrays




What is an Array

An array is a special variable, which can hold more than one value at a time. If you have a list of items (a list of pet names, for example), storing all the petss in single variables .

Initializing Arrays in C#

  int[] amount = new int[40];

Assigning Values to an Array

You can assign values to individual array elements, by using the index number, like:

amount[0] = 100;

You can create and initialize arrays in the same time, as below

  int [] numbers = new int[4]  { 55,  66, 77, 88};

Arrays in c# are simple data structures used to store data items of a specific types and a set of similar type values that are stored in a sequential order. In c#, we can arrange the data in arrays in 3 different ways.

  • Single dimensional arrays
  • Multi dimensional arrays
  • Jagged arrays

In c# arrays can be declared as fixed length or dynamic. Fixed length of arrays can store predefined no.of items while the size of dynamic arrays increases as you add new items to the array.

We access the values of an array by index positions where the array index starts at zero that means 1st item of an array will be at 0th position and the position of last item will be total no.of items-1.

Single Dimensional arrays in C#

Single Dimensional arrays in c# are used to store the data in the form of a row.

Syntax 
  <type>[] <name> = new <type>[size];
Initializing Single Dimensional Arrays in c#

Declaring an array does not initialize the array in the memory. When the array variable is initialized, you can assign values to the array. Array is a reference type, so you need to use the new keyword to create an instance of the array. See the below example,

int[] arr = new int[4];
                (or)
  Int[] arr;
  Arr = new int[3];
                (or)
  Int[] arr = {list of values};

In c# language, an array gets initialized either with the use of new keyword or assignment of values.

Multidimensional Arrays in C#

Multidimensional Array arrays are used to store the data in the form of rows and columns.

Multidimensional Array Syntax in C# 
  
<type>[ , ] <name> = new >type>[rows, columns];
Initializing Multidimensional Arrays in C#

Declaring an array does not initialize the array in the memory. When the array variable is initialized, you can assign values to the array. Array is a reference type, so you need to use the new keyword to create an instance of the array. See the below example,

  int[,] arr = new int[3,4];
            (or)
    Int[,] arr;
    Arr = new int[2,3];
            (or)
    Int[,] arr = {list of values};

Jagged arrays

This is also same as a Multidimensional Array array but in a 2=D array all the rows will have equal no.of columns whereas column size varies from row to row.

Jagged arrays in c# also called as the array of arrays. Because it is a combination of multiple single dimensional arrays with different sizes combined together to form a new array.

Syntax for jagged arrays in C#: 
  [ ][ ]  = new [rows][];

Declaring an array does not initialize the array in the memory. When the array variable is initialized, you can assign values to the array. Array is a reference type, so you need to use the new keyword to create an instance of the array. See the below example,

Initialiazing Jagged Arrays in C# 
 int[][] arr = new int[3][];
                (or)
  Int[][] arr = {list of values};

While declaring jagged arrays first we need to specify no.of rows we want to the array and then painting to each row we must specify the no.of columns to that row.
