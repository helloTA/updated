C# Loops




What is a Loop

A loop is group of statements we want to execute several times until we say stop.

A loop statement allows us to execute a statement or a group of statements multiple times:

C# provides 4 different loops that allows you to execute a block of code repeatedly until a certain condition is met. We call these loops as Conditional loops in c#.


  1. for loop
  2. while loop
  3. do-while loop
  4. foreach loop

For loop in C#

By using a for loop, you can run a statement or a block of statements repeatedly until a specified expression evaluates to false. This would be more useful for iterating over arrays and for other applications. Before made this loop in c# we get to know in advance how many times the loop has to iterate.



for loop syntax in c# 

 for(initializer; condition; iteration)
{
   -- statements.
}

Now let's look at a practial example of for loop in c#


    class ForLoop
{
    static void Main() 
    {
        for (int i = 1; i <= 5; i++)
        {
            Console.Write(i+”  “);
        }
    }
}

If you run the above program it will produce the following output

Output
1 2 3 4 5

While loop

The while loop simply executes a block of code as long as the condition you give it is true.

Syntax

The syntax of a while loop in C# 


    while(condition)
{
   statement(s);
}

Let's look at a sample example of a while loop program in c#

While loop example in c# 

using System;
namespace Loops 
{
   class WhileLoopExample
   {
      static void Main(string[] args)
      {
         /* local variable definition */
         int apples = 10;
         int count = 1;

         /* while loop execution */
         while (count < apples )
         {
            Console.WriteLine("There are  {0} apples in the  bag", count);
            count++;
         }
         Console.ReadLine();
      }
   }
}

If you run the above code it will produce the following output

Output
There are 1 apples in the bag
There are 2 apples in the bag
There are 3 apples in the bag
There are 4 apples in the bag
There are 5 apples in the bag
There are 6 apples in the bag
There are 7 apples in the bag
There are 8 apples in the bag
There are 9 apples in the bag

do-while loop in C#

The function of do-while loop works exactly same as while loop but quite opposite. The do loop evaluates the condition after the loop has executed, which makes execute the code at least once..

The syntax of a do...while loop in C# 

do
{
   statement(s);

}while( condition );

Now let's look at a sample example of do while loop in C#

Example 

using System;
namespace Loops
{
   class ExampleDoWhile
   {
      static void Main(string[] args)
      {
         /*  variable definition */
         int score = 90;
         
         /* do loop execution */
         do
         {
            Console.WriteLine("My current score is: {0}", score);
            score= score + 1;
         } 
         while (score <= 100);
         Console.ReadLine();
      }
   }
}

if you run the above code, it will produce the following output

Output
My current score is: 90
My current score is: 91
My current score is: 92
My current score is: 93
My current score is: 94
My current score is: 95
My current score is: 96
My current score is: 97
My current score is: 98
My current score is: 99
My current score is: 100

Control Statements in C#

control statements are used to control the program specifically loops. Sometimes we want to skip the loop condition or sometimes we want to halt the loop and do something else. In such cases the loop control statements are used.

C# provides two different types of control statements.

  1. Break Statement
  2. continue Statement
Break statement in C#

When the break statement is encountered inside a loop, the loop is immediately terminated and program control resumes at the next statement following the loop.

Syntax for a break statement in C#


break;
Example 

using System;
namespace Loops 
{
   class ExampleBreakStatement
   {
      static void Main(string[] args)
      {
         /*  variable definition */
         int x = 5;
         
         /* while loop execution */
         while (x < 15)
         {
            Console.WriteLine("value of x is: {0}", x);
            x++;
            if (x > 11)
            {
               /* terminate the loop when the x value becomes 11 statement */
               break;
            }
         }
         Console.ReadLine();
      }
   }
}

If you run the above code it will produce the following output

Output
value of x is: 5
value of x is: 6
value of x is: 7
value of x is: 8
value of x is: 9
value of x is: 10
value of x is: 11

Observe, the above output. In our program we said while (x < 15) print the valus, but if the values is 11 then break the program. That's the out stopped printing after the x values becomes 11

Continue Statement in C#

The continue statement in C# works somewhat like the break statement. Instead of forcing the program to terminate,the continue statement will force the program to continue to the next iteration of the loop and skipping any code in between.

Syntax for continue statement in c#


     continue;
Example program for c# continue statement 

using System;
namespace Loops
{
   class ExampleProgram
   {
      static void Main(string[] args)
      {
         /* local variable definition */
         int x = 5;
         
         /* do loop execution */
         do
         {
            if (x == 11)
            {
               /*if the x value becomes 11 skip that statement and continue
               to print the next statements */
               x = x + 1;
               continue;
            }
            Console.WriteLine("value of a: {0}", x);
            x++;
         } 
         while (x < 15);
         Console.ReadLine();
      }
   }
}
Output
value of a: 5
value of a: 6
value of a: 7
value of a: 8
value of a: 9
value of a: 10
value of a: 12
value of a: 13
value of a: 14

Notice, The value 11 is not printed!!!! This is because when the x value becomes 11, the loop skip the value and continued to the next iteration.
