C# Constants




What is a Constant

A constant is an identifier with an associated value which cannot be altered by the program during normal execution â€“ the value is constant

Constants in C# is variable which can not be modified once after declaration, at the time of declaration only it is required to be initialize.

The behaviour of a constant variable is similar to the behaviour of static variable.

Constants are also not associated with the object of the class.

The only difference between static and constant variables is static variable can be modified but not constants.

Defining Constants in C#

Constants are defined using the const keyword.

Syntax
const <data_type> <constant_name> = value ;

There are different types of constants in C#. Let's look at each one of them.

  1. Integer Literals
  2. Character Constants
  3. Floating-point Literals
  4. String Literals

Integer Literals

An integer literal is an integer whose value is directly represented in source code.

An integer literal can be a int, octal, decimal or hexadecimal constant. A prefix specifies the base or radix: 0x or 0X for hexadecimal, 0 for octal, and no prefix id for decimal.

Examples of various types of Integer literals

  70         /* int */
  30u        /* unsigned int */
  54         /* decimal */
  0313       /* octal */
  0x8b       /* hexadecimal */
  31l        /* long */
  70ul       /* unsigned long */

Character Constants

Character literals are enclosed in single quotes. For example, 'h' and can be stored in a simple variable of char type.

A character literal can be a plain character (such as 'd', 'c', 'n' etc..), an escape sequence (such as '\t'), or a universal character (such as '\u02C0').

Example program for Character Literals in C# 

using System;
namespace EscapeChar 
{
   class StringLiteralsExample
   {
      static void Main(string[] args)
      {
         Console.WriteLine("Welcome\tto\t Tutorials\tAcademy\n\n");
         Console.ReadLine();
      }
   }
}

If you run the above example code, it would produce the below output.

Output

Floating-point Literals

A floating-point literal consists of the following:

  1. An integral part
  2. A decimal point
  3. A fractional part
  4. An exponent part
  5. An optional suffix

Both the integral and fractional parts are made up of decimal digits. You can omit either the integral part or the fractional part, but not both. You can omit either the decimal point or the exponent part, but not both.

Examples of floating point literals in C# 

  3.14159       
  314159E-5L

String Literals in C#

A string literal is a sequence of zero or more Unicode characters beginning and ending with an ASCII double-quote character, a Unicode left double-quote character, or a Unicode right double-quote character. Within a string, a sequence of two double-quote characters is an escape sequence representing a double quote in the string. A string constant is of the String type.

  • String literals or constants are enclosed in double quotes "" or with @""
  • A string contains characters that are similar to character literals: plain characters, escape sequences, and universal characters.
