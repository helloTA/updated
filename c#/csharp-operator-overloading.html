C# Operator Overloading




What is Operator Overloading

Where different operators have different implementations depending on their arguments. Operator overloading is generally defined by the language, the programmer, or both.s

Defining multiple behaviours to an operator is known as operator overloading where different behaviour comes into picture depending upon the type of operands i.e., whenever the operand changed operator behaviour in different ways.

For example , ‘+’ a is overloaded operator which works an additional operator when used between numeric operands and works as a concatenation operation when used between 2 string operands or string and numeric operands. 

  10 + 20 
  number + number ⇒ Addition
  string + string   ⇒ Concatenation
  number + string   ⇒ Concatenation

If required, we can also define new behaviours to any existing operator by defining an ‘operator method’ as following.

Syntax 
  
[<modifiers>] static <type> operator <opt> (<operand type definition>)
{
  -- statements
}
  1. Operators methods must be defined as static only.
  2. While defining the operator method we need to specify the return type or type of the result which will come back when the operator is used between the operands.
  3. The name of Operator method is ‘operator’ only.
  4. refers to the operator we wanted to overload.
  5. ‘Operand type definition’ means between which operand do we want to use the operator.
Example program for C# operator overloading

The following program demonistrates the usege of operator overloading 

class Matrix
{
    int a, b, c, d;
    public Matrix(int a, int b, int c, int d)
    {
        this.a = a;
        this.b = b;
        this.c = c;
        this.d = d;
    }

    // Overloading the + operator so that we can use it for adding the 2 matrix's
    public static Matrix operator +(Matrix m1, Matrix m2)
    {
        Matrix obj = new Matrix(m1.a + m2.b, m1.b + m2.b , m1.c + m2.c , m1.d + m2.d);
        return obj;
    }

    // Overloading the - operator so that we can use it for subtracting values of 2 matrix's
    public static Matrix operator -(Matrix m1, Matrix m2)
    {
        Matrix obj = new Matrix(m1.a - m2.b, m1.b - m2.b, m1.c - m2.c, m1.d - m2.d);
        return obj;
    }


    // Overriding the ToString() method inherited from object class so that it returns values associated with the matrix.
    public override string ToString()
    {

        return string.Format("[a:{0} b:{1} c:{2} d:{3}] ", a, b, b, d);
    }

}


class TextMatrix
{
    static void Main()
    {
        Matrix m1 = new Matrix(5, 6, 7, 8);
        Matrix m2 = new Matrix(1, 2, 3, 4);
        Matrix m3 = new Matrix(3, 4, 5, 6);
        Matrix m4 = m1 + m2;
        Matrix m5 = m3 + m4;
        Matrix m6 = m1 - m2;

        Console.WriteLine("Matrix M1 results :" + m1);
        Console.WriteLine("Matrix M2 results :" + m2);
        Console.WriteLine("Matrix M3 results :" + m3);
        Console.WriteLine("Matrix M4 results :" + m4);
        Console.WriteLine("Matrix M5 results :" + m5);
        Console.WriteLine("Matrix M6 results :" + m6);

    }
}

if you run the above example code it will produce the following output

Output
Matrix M1 results : [a:5 b:6 c:7 d:8]
Matrix M2 results : [a:1 b:2 c:2 d:4]
Matrix M3 results : [a:3 b:4 c:4 d:6]
Matrix M4 results : [a:7 b:8 c:8 d:12]
Matrix M5 results : [a:11 b:12 c:12 d:18]
Matrix M6 results : [a:3 b:4 c:4 d:4]

Whenever we try to to print the object of a class using Write or WriteLine methods it will print the values associated with the class internally the ToString() method which is inherited from the Object class get called and the method returns the name of class to which the object belongs and that name gets printed.

As the ToString() method which is inherited from Object class is declared virtual under object class we have a chance of overriding the method as per our requirements so under Matrix class we are overriding the ToString() method to return the values of the matrix in place of class name.

Note:

Substituting variables under a string can be performed under Write and WriteLine methods of Console class as well as format method of string class but the difference is Write and WriteLine will print the values and Format method will return the values.
