Decision Making in PHP




What is Decision Making

If today is Sunday? let's go to movie Else if today is Monday? then let's go to school.

Decision making is about deciding the order of execution of statements based on certain conditions or repeat a group of statements until certain specified conditions are met

Conditional statements used to make decisions.

Conditional statements are used to control the execution of the program.

PHP has 3 different decision-making statements:

1. if else statement

2. else if statement

3. switch statement


What is If...Else Statement in PHP

If you want to execute a piece of code if and only if ; a condition is true and another code if a condition is false, use the if....else statement.

Syntax for if else statement in PHP


 <?php 

 if (condition)
   code to be executed if condition is true;
 else
   code to be executed if condition is false;
?>

Example


 <?php 

  $i=1;
     
  if ($i == 0)
  {
      echo 'i equals 0';
  }
  elseif ($i == 1)
  {
      echo 'i equals 1';
  }
  elseif ($i == 2)
  {
      echo 'i equals 2';
  }
?>

Output

i equal to 0

What is ElseIf Statement in PHP

If person is male? then say He is Male.
Else if person is female? then say she is female.
if not both? then say 'Not indicated'

else if statement is used as extension of "If" structure if the condition fails then it executes another "If" condition to execute the code segment under the "else if" statement

Syntax for elseif statement in PHP


 <?php 

 if (condition)
  code to be executed if condition is true;
elseif (condition)
  code to be executed if condition is true;
else
  code to be executed if condition is false;
?>

Example


 <?php 

  $c = 10;
  $d = 10;
  if ($c > $d)
  {
  echo "c is bigger than d";
  }
  elseif ($c==$d)
  {
  echo "c is equal to d";
  }
  else
  {
  echo "d is smaller than c";
  }
?>
Output
c is equal to d

Switch Statement in PHP

The switch statement is used to avoid long blocks of if..elseif..else code.

If you want to select one of many blocks of code to be executed, use the Switch statement.

Syntax of PHP Switch Statement


 <?php 

 switch (expression)
{
case label1:
  code to be executed if expression = label1;
  break; 
case label2:
  code to be executed if expression = label2;
  break;
default:
  code to be executed
  if expression is different
  from both label1 and label2;
}
?>

Example


 <?php 

 $value = 'AA';
switch($value)
{
  case ('AA'):
      echo "value equals 1";
  case ('BB'):
      if ($value == 'BB'){
          echo "value equals 2";
      }
  case (('AA') || ('CC')):
      echo "value equals 3";
      break;
}
?>

value equals 1
value equals 3
