Strings in C#




Strings in C#

A string is an object of type String whose value is text. which can be stored internally as a sequential collection of char objects. There is no null terminating character at the end of a C# string; therefore a C# string can contain any number of embedded null characters ('\0').

We can even check the length of the string by Length property. The Length property of a string represents the number of Char objects it contains, not the number of Unicode characters. To access the individual Unicode code points in a string, use the StringInfo object.

Creating a Strings in C#

There are different ways to create string object in C#.Let's look at few ways

Assign a string literal to a String variable
Example 
  //string literal and string concatenation
     string wifename, husbandname;
     wifename = "Mary";
     husbandname = "John";
     
     string couple = wifename + husbandname;
     Console.WriteLine("Couple Name: {0}", couple);
Output

The above program will produce the following output

Couple Name: MaryJohn
Using a String class constructor
Example 
  //Using string constructor
   char[] letters = { 'G', 'o', 'o', 'd','n', 'i', 'g', 'h', 't' };
   string greetings = new string(letters);
   Console.WriteLine("Hello,: {0}", greetings);

The above program will produce the following output

Hello, Goodnight
Retrieving a property or calling a method that returns a string
  //methods returning string
     string[] myarray = { "We", "Are", "Tutorials", "Academy" };
     string message = String.Join(" ", myarray);
     Console.WriteLine("Our Message: {0}", message);

The above program will produce the following output

The Message: We Are Tutorials Academy

The complete set of escape sequences is as follows:

  • \' - single quote, needed for character literals
  • \" - double quote, needed for string literals
  • \\ - backslash
  • \0 - Unicode character 0
  • \a - Alert (character 7)
  • \b - Backspace (character 8)
  • \f - Form feed (character 12)
  • \n - New line (character 10)
  • \r - Carriage return (character 13)
  • \t - Horizontal tab (character 9)
  • \v - Vertical quote (character 11)
  • \uxxxx - Unicode escape sequence for character with hex value xxxx
