C# Type Conversion




What is type conversion in C# ?

It is a way of implicitly or explicitly, changing an entity of one data type into another

Type conversion in C# is nothing but converting one type of data to another type. There are 2 types of Type conversions exist in c# language.

Implicit type conversion in C#

Implicit type conversions are basically handled by C# in a type-safe manner.In c#, we see smaller types and lager types or complex types

conversion from smaller to larger integral types and conversion from derived classes to base classes.


Explicit type conversion in C#

Explicit type conversion are handled explicitly by programmers using the predefined functions. Explicit conversions require a cast operator.

Example: 

static void Main(string[] args)
    {
        double price = 8980.09;
        int cost;
        // cast double to int.
        cost = (int)price ;
        Console.WriteLine(cost);
        Console.ReadKey();
    }

From this example casting done explicitly from decimal type to integer type.

C# Type Conversion Methods

Let's look at another example to understand how type conversion works in C#.

C# type conversion example for double to int 

using System;
namespace TypeConversionApplication 
{
   class ExplicitConversionExample 
   {
      static void Main(string[] args) 
      {
         double d = 7873.74; 
         int i;
         
         // casting double to int.
         i = (int)d;
         Console.WriteLine(i);
         Console.ReadKey();
      }
   }
}

Output
7873

If you run the above example it will produce 7873 as output because it converted the double value to int. You can also try the same example for float to int as well.

ToInt32 Converts a type to a 32-bit integer.
ToInt64 Converts a type to a 64-bit integer.
