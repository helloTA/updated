PHP Constants




What is constant?

A constant is an identifier for a data value which cannot be changed later.

Syntax 

 <?php 

define (name, value)
?>

Parameters:

name: Specifies the name of the constant

value: Specifies the value of the constant

Example: 

 <?php 

define (“PI”, 3.14);
echo PI;
?>

Constants are Global

Constants are automatically global and can be used across the entire PHP script .

The example below uses a constant inside a function, even if it is defined outside the function:


Example 

 <?php 

define("PI", "3.14");

function myExamle() {
    echo PI;
}
 
myExamle ();
?>   



Output: 3.14
