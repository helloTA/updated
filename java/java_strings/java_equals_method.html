Java String equals() Method




Java String equals() Method

Java string equals() method is used to compare two string objects to check if both are equal or not. For example to check if the lastname of two twin brothers is same or not. If both are same it returns TRUE otherwise it returns FALSE.

It takes string as an argument and returns boolean as a result

Java String equals() Method Syntax

public boolean equals(string obj)

Example

        /*
      String equals Example
      This example shows how to check if two strings are equal or not.
    */


   public class EqualMethodExample {

   public static void main(String args[]) {
       
       // check if the elder brother's lastname is equals to yourger brother's lastname
      String elder_brother_lastname = new String("zuckerberg");
      String younger_brother_lastname = new String("zuckerburg");
      
      boolean result1;

      result1 = elder_brother_lastname.equals(younger_brother_lastname );
      System.out.println("Match is = " + result1 );
      System.out.println("The lastnames of both brothers is not same" );
      
      
        // check if the elder sister's lastname is equals to yourger sister's lastname
      String elder_sister_lastname = new String("Oliva");
      String younger_sister_lastname = new String("Oliva");
      
      boolean result2;

      result2 = elder_sister_lastname.equals(younger_sister_lastname );
      System.out.println("Match is = " + result2 );
      System.out.println("The lastnames of both sister are same" );

   }
}

It produces the following output

Output 
  Match is false
  The lastnames of both brothers is not same
  Match is true
  The lastnames of both sisters is same
