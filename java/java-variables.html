Java Variables




What is a Variable

In programming, a variable is a value that can change, depending on conditions or on information passed to the program. Typically, a program consists of instruction s that tell the computer what to do and data that the program uses when it is running.
Java Variables

Variables are type of value that are capable of storing the data in it. Each variable has a specific type, which determines the size and layout of the variable's memory; the range of values that can be stored within that memory; and the set of operations that can be applied to the variable.

Note: In java, You must declare variables before going to use them.

Syntax 
data type variable [ = value][, variable [= value] ...] ;

Here data type is one of Java's datatypes and variable is the name of the variable. To declare more than one variable of the specified type, you can use a comma-separated list.

Following are valid examples of variable declaration and initialization in Java: 

int a, b, c;            // Declares three ints, a, b, and c.
int a = 100, b = 22;  // Example of initialization
byte B = 12;          // initializes a byte type variable B.
double pi = 3.14159;  // declares and assigns a value of PI.
char a = 'v';         // the char variable a is initialized with value 'v'

This chapter will explain various variable types available in Java Language. There are three kinds of variables in Java:

  1. Local variables
  2. Instance variables
  3. Class/static variables

Local variables:

If a variable that is declared inside the method is called local variable. The local variable in java is accessible only within the specific method or class.

Java Local Variable Example

  public class Example{ 
   public void myRank(){
      int rank = 1;         // here rank is local variable
      rank = rank + 3;      // the rank variable is accessible only within the myRank method
      System.out.println("My Rank is : " + rank);
   }
   
   public static void main(String args[]){
      Example example = new Example();
      example.myRank();
   }
}

When you run the above example, it will produce the following output

Output 
  My Rank is 4

Instance Variables in java

An instance variable is a variable defined in a class (i.e. a member variable), for which each instantiated object of the class has a separate copy, or instance. An instance variable is similar to a class variable.

Example

import java.io.*;

public class Student{
   // this instance variable is visible for any child class.
   public String name;
   
   // marks  variable is visible in Student class only.
   private int marks;
   
   // The name variable is assigned in the constructor. 
   public Student (String stName){
      name = stName;
   }

   // The marks variable is assigned a value.
   public void setMarks(int stMarks){
      marks = stMarks;
   }
   
   // This method prints the Student details.
   public void printSt(){
      System.out.println("M.r  " + name + " got " + marks + " marks " );
      System.out.println("M.r  " + name + " got " + marks + " marks " );
      
   }

   public static void main(String args[]){
      Student stOne = new Student("John");
      stOne.setMarks(90);
      stOne.printSt();
      
      Student stTwo = new Student("Michel");
      stTwo.setMarks(65);
      stTwo.printSt();
   }
}
Output 

M.r  John got 90 marks 
M.r  Michel got 65 marks

Class or static variables in java:

If avariable that is declared as static is called static variable. It cannot be local.

Static variables are initialized in different ways.

  1. Static variables gets initialized by the time of class is being loaded.
  2. Static variables gets initialized before an object of that class created.
  3. Static variables in a class are initialized before any static method of the class runs.

import java.io.*;

public class MathClass{

   public static void main(String args[]){

   }

Output
