Java Decision Making




What is Decision Making

When you are going to write some program it will be always necessary that you will have to put some code which has to be executed only when a specific condition is satisfied. These conditions are evaluated by the computer itself when the input is given by the user.

Java provides Four different types of conditional statements which are as follows.

  1. if Statement
  2. if-else Statement
  3. Nested if else
  4. Switch Statement

If you are confused on, which one to use!!! All the above four methods can be used for decision making but their application will vary, you will feel comfortable after making a few programs using them.

if Statement

As the word itself is clarifying its meaning, it is used to check a condition and perform a set of statements according to the result of the condition checked

If condition generally works with the boolean expression. Syntax: 

   
 if(condition)
 {
 ....
 ....               // TRUE Block Statements
 }

In the above syntax, the conition is a boolean expression. If the expression is true then it executes the block of code inside the curly bracs { }

Example

int age = 25;
if (age > 20)
{
    System.out.println("Yes, age is Greater than 20");
}

In the above example, the java if condition would prints the statement "Yes, age is Greater than 20" if the condition is really true. In our case, it is true. So the output would be like below.

Output 
Yes, age is Greater than 20
if-else Statement
If else statement works similar to the java if statement. But the only difference is, if the provided condition is wrong or false it we execute someother statement in the else block. For example, if the age is > 20 print He is elder else printHe is not elder. Thats how the if else statement works.

If condition generally works with the boolean expression. Syntax: 

   
 if(condition)
 {
 ....
 ....                   // TRUE Block Statements
 }

else {
  .....
  .....                 // FALSE Block Statements
}
In the above given syntax "condition" means any relational condition or any mathematical expression. If the condition given in the brackets (parenthesis) is TRUE then the statements within the BRACES executed. The statements given in BRACES after the "else" part are executed when the condition is false (not satisfied)

Example

int age = 18;
if (age > 20)
{
    System.out.println("He is Elder");
}
else
{
System.out.println("He is not Elder");
}

The above example prints He is not Elder because the condition is FALSE

Output 
He is not Elder

Example Program

int var1=100, var2=200;
if(var1 > var2)
 {
  System.out.println("Variable 1 is bigger ");
 }
else
 {
  System.out.println("Variable 1 is smaller");
 }
}
Explanation :

There are two variables in the above program which are having some values. In the given example the value of val1 is smaller then var2, so the condition for if is false. According the program is going to print the output. You guessed right. The output will be "Variable 1 is smaller".

Nested IF Statements

There are times when it will be required to check more than one conditions and according to their collective results would like to perform various tasks. We can use IF statements within another if statements for any number of times required. Although there is not any separate syntax for if else statements.

Syntax 
if(condition1)
{
  if(condition2)
  {
  ...
  //statements
  }
  else
  {
  ...
  //statements
  }
}
else
{
...
//statements
}

Let's take a look at example program for nested if statement


bool condition = true;
bool check= true;
if (condition)
{
if (check)
    {
      Console.WriteLine("The nested variable is set to true.");
    }
    else
    {
      Console.WriteLine("The nested variable is set to false.");
    }
}
else
{
   Console.WriteLine("The variable is set to false.");
}
Output 
  The nested variable is set to true.

Switch Statement

Switch statement in java is used to check a good number of conditions and will work accordingly. Switch is a method which is going to take an integer parameter and depending upon that parameter will do the given operations.

Syntax 
  switch(value)
  {
   case value:
   statements;
   break;
   case value:
   statements;
   break;
   default;
   statements;
   break;
  }

As the syntax shows that within parenthesis of switch we will always have to give some value as an integer and depending upon the value of that expression switch will select the equivalent case. After executing the case statements if the break is missing then all the statements below that case are also executed. If there is no case value which matches the value in the switch then statements in the default block will be executed.

Example 

/* Assume the days are as follows:
     Sunday - 1
     Monday - 2
     Tuesday - 3
     Wednesday -4
     Thursday -5
     Friday -6
     Saturday -7 */

     // Now, using the switch statement, we have to print the day based on the given
     // condition. i.e value. If value is 1, it should print Sunday, if value is 2 it should 
     // print Monday ...etc 

int today = 3;
switch (today)
{
  case 1:
       Console.WriteLine("To day is Sunday");
       break;
  case 2:
       Console.WriteLine(“Today is Monday");
       break;
  case 3:
         Console.WriteLine(“Today is Tuesday");
         break;
  case 4:
         Console.WriteLine(“Today is Wednesday");
         break;
  case 5:
         Console.WriteLine(“Today is Thursday");
         break;
  case 6:
         Console.WriteLine(“Today is Friday");
         break;
  case 7:
         Console.WriteLine(“Today is Saturday");
         break;

   default:
       Console.WriteLine("Day is not available!");
       break;
}

In the above example, the given day is today which has value 3. In the switch cases if the case is 3 it should print the third case which is Today us Tuesday

Output 
  Today is Tuesday
