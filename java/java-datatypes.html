Java Data Types




What is a Data type

A data type in a programming language is a set of data with values having predefined characteristics. The language usually specifies the range of values for a given data type, how the values are processed and how they are stored.

Data Types in Java

Data types in Java are actually derived from the programming language ‘C’. In java, Data types are interlinked with variables where Variables are the one which stores the data and the operating system allocates memory when variable creation happens.

Operating System decides what can be stored in the allocated memory depends on the datatype of a variable. We can store strings, integer, decimal and others types of variables by assigning different data types to them.


There 2 two data types available in Java:

  1. Primitive Data Types
  2. Reference/Object Data Types
Primitive Data Types:

Primitive Data types form the backbone to represent the properties/attributes of the objects. Any form of the data can be stored by using the primitive data types in Java. Java language supports 8 primitive data types.

types Description Capacity
byte(8 bit) Used to save space in large arrays, mainly in place of integers, since a byte is four times smaller than an int. -128 to 127
short(16 bit) Used to save memory as byte data type. A short is 2 times smaller than an int -32768 to 32767
int(32 bit) Int is generally used as the default data type for integral values unless there is a concern about memory -231 to 231-1
long(64 bit) Used when a wider range than int is needed --
float(32 bit IEEE ) Used to save memory in large arrays of floating point numbers. 0 to 65535
double Used as the default data type for decimal values, generally the default choice 0 to 264-1
boolean used for simple flags that track true/false conditions true/false
char(16-bit Unicode character) used to store any character '\u0000' (or 0) to'\uffff' (or 65,535)

Let us look at some examples of primitive data types in java

Examples:

  • byte a = 123, byte b = -90
  • short s = 3423, short r = -20000
  • int a = 4546, int b = -200000
  • long jimmy= 100000L, int b = -200000L
  • float distance= 51.5f
  • double d1 = 123.4
  • boolean isMale= true
  • char ch='A'

Now let us understand about the reference data types in java

Reference Data Types in Java:

A reference variable is declared to be of a specific type and that type can never be changed. Reference variables can be declared as static variables, instance variables, method parameters, or local variables.

Class objects and various type of array variables come under reference data type. Default value of any reference variable is null. A reference variable can be used to refer to any object of the declared type or any compatible type.
