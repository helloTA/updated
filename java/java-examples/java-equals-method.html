Java equals() Method




Java equals() Method

It checks whether the two arguments are equal or not. If both are equal it returns TRUE else FALSE.

Syntax for java equals() Method

    public boolean equals(Object x)

For example see the below example

  int marks = 65;
  int marks_required_to_get_first_class = 75
  System.out.println(marks.compareTo(marks_required_to_get_first_class));

  // In the above example the student got 65 marks
  // But to get first class certificate he/she needs to score 75.
  // using the java equals() method the above code checks if the marks equals to 75 or not.
  // Since the above condition is not met, it returns FALSE.

Example


 /* The below example returns the student results.
  The student got different marks in different subjects.
 but the passmark is 35. If the marks in specific subject is equals to
 the passmark_for_every_subject then it returns TRUE otherwise it returns FALSE. */ 


  public class Test{ 

   public static void main(String args[]){
      Integer mymarks_in_english = 55;

      Integer mymarks_in_chemistry = 20;

      Integer mymarks_marks_in_programming = 65;

      Integer mymarks_in_algebra = 21;

      Integer passmark_for_every_subject = 35;


     // If mymarks_in_english is equals to passmark_for_every_subject then it returns TRUE
      System.out.println(mymarks_in_english.equals(passmark_for_every_subject));  

     // If mymarks_in_chemistry is equals to passmark_for_every_subject then it returns TRUE
      System.out.println( mymarks_in_chemistry.equals(passmark_for_every_subject));

     // If mymarks_in_programming is equals to passmark_for_every_subject then it returns TRUE
      System.out.println(mymarks_marks_in_programming.equals(passmark_for_every_subject));

     // If mymarks_in_algebra is equals to passmark_for_every_subject then it returns TRUE
      System.out.println(mymarks_in_algebra.equals(passmark_for_every_subject));
     }
}
