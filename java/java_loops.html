Java Loops




What is a Loop

A loop is group of statements we want to execute several times until we say stop.

A loop statement allows us to execute a statement or a group of statements multiple times:

Java provides 3 different loops that allows you to execute a block of code repeatedly until a certain condition is met.

  1. for loop
  2. while loop
  3. do-while loop

for loop in java

By using a for loop, you can run a statement or a block of statements repeatedly until a specified expression evaluates to false. This would be more useful for iterating over arrays and for other applications.


Java for loop syntax


 for(initialization; Boolean_expression; update)
{
   //Statements
}
How does the for loop in java works?
  1. The initialization step will be executed only once.
  2. Boolean_expression: If expression is true, the body of the loop is executed. If it is false, the body of the loop does not execute and flow of control jumps to the next statement.
  3. After the body of the for loop executes, the flow of control jumps back up to the update statement. This statement allows you to update any loop control variables. This statement can be left blank, as long as a semicolon appears after the Boolean expression.
  4. The Boolean expression is now evaluated again. If it is true, the loop executes and the process repeats itself (body of loop, then update step, then Boolean expression). After the Boolean expression is false, the for loop terminates.

Java For loop Example

  public class ForLoop {

   public static void main(String args[]) {

      for(int x = 5; x < 15; x = x+1) {
         System.out.print("value of x : " + x );
         System.out.print("\n");
      }
   }
}

The above java for loop program will produce the following output

value of x : 5
value of x : 6
value of x : 7
value of x : 8
value of x : 9
value of x : 10
value of x : 11
value of x : 12
value of x : 13
value of x : 14


While loop in Java

The while loop simply executes a block of code as long as the condition you give it is true.

Java While Loop Syntax

while(expression)
{
   //Statements
}

if the expression result is true, then the actions inside the loop will be executed. This will continue as long as the expression result is true.

Java While Loop Example

public class WhileLoop {

 public static void main(String args[]) {
    int a = 5;

    while( a < 11 ) {
       System.out.print("value of a : " + a );
       a++;
       System.out.print("\n");
    }
 }
}

If the a value is less than 11 than it will increase the value of a by 1 and prints the a value until the expression becomes false. In this case, until a value becomes more than 11

Output 
value of a : 5
value of a : 6
value of a : 7
value of a : 8
value of a : 9
value of a : 10


do while loop in java

The function of do-while loop works exactly same as while loop but quite opposite. The do loop evaluates the condition after the loop has executed, which makes execute the code at least once..

Java do while loop syntax 
do
{
   //Statements
}while(expression);

If the Boolean expression is true, the flow of control jumps back up to do, and the statements in the loop execute again. This process repeats until the Boolean expression is false.

Example program for java do while loop

  public class doWhile {

   public static void main(String args[]){
      int a = 1;

      do{
         System.out.print("value of a : " + a );
         a++;
         System.out.print("\n");
      }while( a < 5 );
   }
}
Output 
value of a : 1
value of a : 2
value of a : 3
value of a : 4


Java break Keyword:

The break keyword is used to stop the entire loop. The break keyword will stop the execution of the innermost loop and start executing the next line of code after the block.
Syntax for java break statement

The syntax of a break is a single statement inside any loop: 

break;

Java continue Keyword:

It causes the loop to immediately jump to the next iteration of the loop.

In a for loop, the continue keyword causes flow of control to immediately jump to the update statement.

In a while loop or do/while loop, flow of control immediately jumps to the Boolean expression.

java continue statement syntax:

The syntax of a continue is a single statement inside any loop: 

continue;
