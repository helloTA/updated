Java Arrays

What is an Array

Imagine a situation where you want to store 1000 employee records. As you might already know that, in java we first need to create the variables we want to use in programming. So in that case, if you want to create record for 1000 employees how do you do that?. Do you just declare 1000 student records like below?

  int employee1_id = 1;
  int employee2_id = ;
  int employee3_id = 3;
  int employee4_id = 4;
  int employee5_id = 5;
  int employee6_id = 6;
  int employee7_id = 7;
  .
  .
  .
  .
  .
  .
  .
  int employee1000_id = 1000;

If you declare like above, that is not the best practice. However in such kind of cases, java introduces a nice concept called Arrays. You might be doubting what does array do in that case? See the below the implementation of creating 1000 employee records. 

int[] employeeArray = new int[1000];

The above employeeArray allows you to store 1000 employee records of same int type.

Arrays in Java

Java Array is a collection of similar type of elements that have contiguous memory location. It is a data structure where we store similar elements. We can store only fixed set of elements in a java array. Array in java is index-based, the first element of the array is stored at 0 index.

How to create Arrays in Java

To create an array in java you have to use new keyword.

Syntax for Creating Arrays in Java

myarrayName = new dataType[arraySize];

In the above array declaration, myarrayName is the name of the arrays which is used to refer the data members in that array.

You can also declare the same array in another way as follows:

  int[] myarrayName = {value0, value1, ..., valueN};

In some cases, if you are not sure about the size of an array you can declare an array as follows:

  int[] myarrayName = [];   // if the size is unknown

Example of Comparing Two Int Arrays

  /*
  Compare Two Java int Arrays Example
  This java example shows how to compare two integer arrays for equality using
  Arrays.equals method.
*/
 
import java.util.Arrays;
 
public class CompareIntArraysExample {
 
  public static void main(String[] args) {
    //create int arrays
    int[] myArray1 = new int[]{1,11,111,1111};
    int[] intArray2 = new int[]{1,11,111,1111};
   
    /*
      To compare two int arrays use,
      static boolean equals(int array1[], int array2[]) method of Arrays class.
     
      It returns true if both arrays are equal. Arrays are considered as equal
      if they contain same elements in same order.
    */
   
    boolean result = Arrays.equals(myArray1,myArray2);
    System.out.println("Are the two int arrays are equal? : " + result);
  
   
  }
}
Output 
  Are the two int arrays are equal ? : true

Java Compare Two char Arrays Example

  This java example shows how to compare two char arrays for equality using
  Arrays.equals method.
*/
 
import java.util.Arrays;
 
public class CompareCharArraysExample {

      To compare two char arrays use,
      static boolean equals(char array1[], char array2[]) method of Arrays class.

    boolean result = Arrays.equals(mycharArray1,mycharArray2);

  }

Output 
  Are two char arrays equal ?: false
