CSS Selectors

What is a CSS Selector

CSS selector is is property which is used target a specific element and apply the neccessary styles to that element. Imagine, there 100 paragraph elements in a webpage and you want to target 1 specific paragraph element change the font-color of that paragraph, how do you do that?. Thats where that CSS Selectors comes in

CSS selectors allow you to select and change the HTML elements on a web page very easily.

CSS selectors are used to find or select a specific HTML elements based on their id, class, type, attribute, and more.

CSS Selectors are different types which we can use in different times

The most common CSS selectors are as follows.

  1. The Element Selector
  2. The id Selector
  3. The Class selector
  4. Descendant Combinator
  5. Child Combinator
  6. General Sibling Combinator
  7. Adjacent Sibling Combinator

Let's look at each of them

CSS Element Selector

The css element selctor is used to target a specific element in a web page and apply the style. But when you have many selectors with the same name, css element selector is not a good choice.

Example

Let’s see what the CSS code for this looks like:

h4 {

 color: green; /* target h4 element and apply the style*/
}

p { 

 color: red;     /* target the p element and apply the text color as red*/
}

So, if we apply this selector to the following HTML:

  <p>Targeting this paragraph and apply the style.</p>

</body>

CSS id Selector

CSS id selector is used to overcome the problem raised in element selector. When you have many elements with the same name and you want to target each of the css elements it would be very difficult.

So the css id selector is used to target each and every element or any no of elements in a web page by giving them uniq ids. Imagine, if you have 10 paragraphs you can give 10 different uniq names and target them using the id selector.

Id selector is starts with # symbol followed by the selector name

For example take look at the below example

    <p id="mypara1"> This is paragraph one</p>
    <p id="mypara2">This is paragraph two </p>

This is how you can give the selector name. See the below css selector code to to understand how you can target that mypara1 and mypara2 ids

Example

Let’s see what the CSS code for this looks like:

#mypara1 {
color: green;
}
#mypara2 {
color: red;
}
Note:

Selector names must be unique


<body>

<p id="mypara1">Hello World!</p>
<p id="mypara2">This is effected by the selector</p>

</body>

CSS class Selector

Imagine, you have 10 elements and you want to apply the same styles to all the 10 elements. Of course, you can give 10 different uniq id names and apply the styles. But thats not a good practice!. Why? What if you have 1000? Do you give 1000 names.

That's where the css class selector comes in. What it does is it applys the same style for any no of elements which comes under the same class.

Example

Let’s see what the CSS code for this looks like:


.myclass {
    text-align: center;
    color: green;
}


  <body>

<div class="myclass">
    <p>This is paragraph</p>
    <h2>This is heading</h2>
    <p>Another paragraph</p>
</div>

</body>

If you notice the above code, I put 2 paragraphs and 1 head tag under one class called myclass. And I applied the same style for all the three elements in a single shot.

The CSS class syntax is . Dot symbol followed by the class name

Always remember the class name can not be start a number

CSS Descendant Combinator

The descendant selector or, more accurately, the descendant combinator lets you combine two or more selectors so you can be more specific in your selection method.

Example:

The CSS code for this looks like:

#myContainer .box {
   float: left;
   padding-bottom: 15px;
color:red;
}

p {
color: blue;
}

So, if we apply this selector to the following HTML:

 <body>

<div id="myContainer">
  <div class="box"> First Div</div>

  <div class="box-2">Second Div</div>
</div>

<div class="box">Third Div</div>

<p><b>NOTE:</b> The style is applied only to the immediate descedant child of myContainer. i.e the first Div</p>
</body>

This declaration block will apply to all elements that have a class of box that are inside an element with an ID of myContainer.

Note: The .box element doesn’t have to be an immediate child: there could be another element wrapping .box, and the styles would still apply.

CSS Child Combinator

A selector that uses the child combinator is similar to a selector that uses a descendant combinator, except it only targets immediate child elements:

General Sibling Combinator

A selector that uses a general sibling combinator matches elements based on sibling relationships. That is to say, the selected elements are beside each other in the HTML.

h2 ~ p {
   margin-bottom: 20px;
   color:green;
}

This type of selector is declared using the tilde character (~). In this example, all paragraph elements (<p>) will be styled with the specified rules, but only if they are siblings of <h2> elements. There could be other elements in between the <h2>and<p>, and the styles would still apply.

Example

The CSS code for this looks like:


h2 ~ p {
   margin-bottom: 20px;
color: green;
}

<p>Paragraph one</p>

<p>Paragraph three.</p>

Adjacent Sibling Combinator

A selector that uses the adjacent sibling combinator uses the plus symbol (+), and is almost the same as the general sibling selector. The difference is that the targeted element must be an immediate sibling, not just a general sibling.

This example will apply the specified styles only to paragraph elements that immediately follow other paragraph elements. This means the first paragraph element on a page would not receive these styles. Also, if another element appeared between two paragraphs, the second paragraph of the two wouldn’t have the styles applied.

Example

Let’s see what the CSS code for this looks like: 
p + p {
   text-indent: 1.5em;
   margin-bottom: 0;
}

So, if we apply this selector to the following HTML:

<h2>My Title</h2>
<p>Paragraph one.</p>
<p>Paragraph two.</p>
<p>Paragraph three</p>

<div class="box">
  <p>Paragraph four.</p>
  <p>Paragraph five.</p>
</div>
