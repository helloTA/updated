CSS Backgrounds




CSS Background

CSS background properties are used to style the background effects of an element or a web page.

CSS has five different background properties we can use. Let's look at each of them

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position

CSS Background Color

The background-color property specifies the background color of an element. It is mainly used when we need to change or set the background-color of a web page or an element.

Example
<!DOCTYPE html>
<html>
<head>
    <style>
    body {
        background-color: yellow;
    }
 </style>
</head>
<body>

    <h1>Home Page</h1>
    <p>Welcome to Tutorials Academy.</p>

</body>
</html>

When you run the above code, it will produce the following output with the background-color as yellow

You can actually use the HEX values to set the CSS colors

a HEX value - like "#ff0000" an RGB value - like "rgb(255,0,0)" a color name - like "red"

Let's see an example to understand more about setting colors in CSS using HEX values

h1 {
    background-color:#ff0000;
}

p {
    background-color: #00ff00;
}

div {
    background-color:  #0000ff;
}

CSS Background Image

The CSS background-image property is used to set a specific image as a background, instead of styling it . We need to give the file or image name

By default, the image is repeated so it covers the entire element.

Example
<!DOCTYPE html>
<html>
<head>
<style>
body {
    background-image: url("http://p1.pichost.me/i/20/1433003.jpg");
}
/* You can also set your image as background from your computer,
 for example like this 
 background-image: url("myfolder/images/wall.png");
 */
</style>
</head>
<body>

<h1> Welcome to Tutorials Academy</h1>

</body>
</html>

CSS Background-repeat

Specifies how a background image will repeat itself within a box.

Possible Values

inherit

repeat (default) - tiled, repeating the image both horizontally and vertically.


repeat-x - repeating the image horizontally only.


repeat-y - repeating the image vertically only.


no-repeat - not repeating the image at all, showing just one instance. 


  #title { background-repeat: repeat-x; }

Let's look at an example of how we can change the css background with the background-image and background-repeat property

Example

   body {
    background-image: url(" http://images.mid-day.com/images/2014/feb/Chhota-Bheem.jpg");
    background-repeat: no-repeat;
    background-position: right top;
    margin-right: 200px;
}
