What is CSS Text Formating

Every style on of text on the web page can be styled in different ways. The text can have many formats This text is styled with some of the text formatting properties. The heading uses the text-align, text-transform, and color properties.

Text Color

CSS text color is used to style the text with different colors. Imagine, you want to style the text on a web page with different colors. This is where you can the text colors. See the below web page for example

We CSS we can set colors in three different ways. Let's look at each of them.

HEX Value: For Example "#00ff00;"

RGB Value: For Example"rgb(255,0,0)"

Color name:For Example "green"

The default color for a page is defined in the body selector.

Note:

If you define the color property, you must also define the background-color property.

Example

Let's take a look at an example to understaind how css color property works.

body {
    color: red;
}

h1 {
    color: #00ff00;
}

h2 {
color: orange;
}


p.ex-two {
color:blue ;
}

p {
color: black; }

If you run the above example the body color will changed to red and h2 color will be orange and paragraph color will be black and others respectively.

Text Alignment

CSS text alignment property is used to place the text on the web page in right order. Like I say, you can align the text wherever you want to!. That being said, you can align the text in center or left or right or justify.Each alignment has different meaning

The text-align property is used to set the horizontal alignment of a text.

Text can be centered, or aligned to the left or right, or justified.

When text-align is set to "justify", each line is stretched so that every line has equal width, and the left and right margins are straight

Take a look at the below example to understand these four css text alignments


Example

h1 {
    text-align: center;
}

p.date {
    text-align: right;
}

p.justify {
    text-align: justify;
}

p.left {
text-align: left;
}

Text Decoration

CSS text-decoration property is used to decorate the text or remove the decoration for the ext. CSS text-decoration property is mostly used to set or remove the underlines on the web page.

In many case, when you wish to have no decoration for the text you can simply say text-decoration: none;. That makes the text to have no decoration .

See the below example for more understanding

Example

  h1 {
    text-decoration: overline;
}

h2 {
    text-decoration: line-through;
}

h3 {
    text-decoration: underline;
}

CSS Text Transformation

CSS text-transformation is used to style the text on the web page either to uppercase, or lowercase, or capitalize. There are some cases where you want to turn all the text into uppercase, or lowercase in that case youc an use the csstext-transformation property

The text-transform property is used to specify uppercase and lowercase letters in a text.

Example

 p.uppercase {
    text-transform: uppercase;   /* Transforms all text to UPPERCASE*/
}

p.lowercase {
    text-transform: lowercase;    /* Transforms all text to UPPERCASE*/
}

p.capitalize {
    text-transform: capitalize;   /* Transforms every letter in a word to capital*/
}
