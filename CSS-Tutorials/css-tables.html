CSS Tables




CSS Tables

Table is a way of representing the data in table format.

Tables are generally created using the HTML elements, but we can style them using CSS. Every table will have a table heading and table rows and columns. The data in tables are represented using rows and columns

HTML tables are created using the HTML <table> tag. You combine this tag with other tags that define each row and column within your table.

For example take a look at the below table, this is a sample table which contains some rows and columns

Table Header 1 Table Header 2
Table cell 1 Table cell 2
Table cell 3 Table cell 4

Table Borders

Table border is used to specify the border in a HTML table. The property we use is border

The example below specifies a black border for <table>, <th>, and <td> elements:

Example

table, th, td {
   border: 2px solid red;
}

Collapse Borders

The css border-collapse property is used to set whether the table borders are collapsed into a single border or seperated

The example HTML table would look like as follows

  <table>
  <tr>
    <th>Student Name</th>
    <th>Marks</th>
  </tr>
  <tr>
    <td>Michel</td>
    <td>64</td>
  </tr>
  <tr>
    <td>Joy</td>
    <td>56</td>
  </tr>
</table>

Let's take a look at how the css would look like for the above HTML table

  table {
    border-collapse: collapse;
}

table, td, th {
    border: 3px solid red;
}

The above example would apply the cssborder-collapse to the table and sets its border as red with 3px

Table Width and Height

CSS width and height property is used to define the height and width of a table

The example below sets the width of the table to 80%, and the height of the <th> elements to 40px:

The example HTML table would look like this:

Example

<table>
  <tr>
    <th>Company</th>
    <th>Founder</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>Facebook</td>
    <td>Mark Zuckerberg</td>
    <td>USA</td>
  </tr>
  <tr>
    <td>Bill Gates</td>
    <td>Microsoft</td>
    <td>USA</td>
  </tr>
  <tr>
    <td>Dropbox</td>
    <td>Drew Houston</td>
    <td>USA</td>
  </tr>
  <tr>
    <td>Infosys</td>
    <td>Narayana Murthy</td>
    <td>India</td>
</tr>
<tr>
    <td>TATA</td>
    <td>J.D Tata</td>
    <td>India</td>
</tr>
</table>

Let's look at the how the css code would look like for the above table using the csswidth and height property

table, td, th {
    border: 1px solid black;
}

table {
    width: 80%;
}

th {
    height: 40px;
}
