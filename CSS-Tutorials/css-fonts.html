CSS Fonts




CSS Fonts

CSS Fonts are used to style the font of html elements. Like font-size, font-color, font-family etc..

when they think “font” are thinking of the font face. The font face is the actual typeface that is used for the type. But technically, the font is everything about the type from the weight (boldness), size, line height, and so on

CSS has two different types of font familes.

  1. Generic Family
  2. font family
Generic Family

If the user agent viewing your Web page does not have access to the font you have chosen, it will use a different font instead. If you indicate which generic font family you'd like the user agent to use, you can have a little more control over your Web page.

The generic fonts are:

  1. cursive
  2. fantasy
  3. monospace
  4. sans-serif
  5. serif

Font Family

The font family of a text is set with the font-family property.

Some sample font families are Times New Roman, Bodoni, Garamond, Minion Web, ITC Stone Serif, MS Georgia, Bitstream Cyberbit The 'font-family' property Syntax 

    [[ <family-name> | <generic-family> ] [, <family-name>| <generic-family>]* ] | inherit
Example
    body { font-family: Gill, Helvetica, sans-serif }
<family-name>:

The name of a font family of choice.

<generic-family>:

The last value is a generic family name. The following generic families are defined:

  1. 'serif' (e.g., Times)
  2. sans-serif' (e.g., Helvetica)
  3. 'monospace' (e.g., Courier)

Example

<!DOCTYPE html>
<html>
<head>
<style>
p.serif {
    font-family: "Times New Roman", Times, serif;
}

p.sansserif {
    font-family: Arial, Helvetica, sans-serif;
}
</style>
</head>
<body>

    <h1>CSS font-family</h1>
    <p class="serif">Welcome to Tutprials Academy. This is Times New roman font </p>
    <p class="sansserif">This is a paragraph, shown in the Arial font.</p>

</body>
</html>

Font Style

The font style property is used style the fonts. Few common font styles are as follows.

normal Text will be shown in normal style


italicText will be shown in italic style

obliqueIt is simialr to oblique the text name is leaning.

font-family Syntax 
font-style: normal | italic | oblique | inherit
Example 
    h1, h2 { font-style: italic; }
    p {font-style: normal ;}

Example

p.normal {
    font-style: normal;
}

p.italic {
    font-style: italic;
}

p.oblique {
    font-style: oblique;
}

Font Size

The font-size property sets the size of the text. You can set the font-size to be either big or small etc..

The font-size value can be an absolute, or relative size.

Absolute size:

Absolute size sets the text to a specified size. Absolute size is useful when the physical size of the output is known

Relative size:

Relative size sets the size relative to surrounding elements Allows a user to change the text size in browsers

Example 
h1 {
    font-size: 20px;
}

h2 {
    font-size: 40px;
}

p {
    font-size: 10px;
}
