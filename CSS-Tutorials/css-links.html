CSS Links

Every link we see on a web page, can be styled into different colors. We can style a link in blue color or red color or black color etc..

Links can be styled with any CSS property (e.g. color, font-family, background, etc.).

Example

  a {
    color: #00ff00;
}

Example 2

  a {
    color: #ff0000;

We can also style the links based on the based on the state of a link, if it visited apply one style, if not visited apply another style

We have four different types of link states in CSS

a:link - a normal, unvisited link

a:visited - a link the user has visited

a:hover - a link when the user mouses over it

a:active - a link the moment it is clicked

Target Attribute Values _blank:

Opens the linked document in a new window or tab

_self:

Opens the linked document in the same frame as it was clicked (this is default)

_parent:

Opens the linked document in the parent frame

_top:

Opens the linked document in the full body of the window

Example

/* selected link */
a:active {
    color:  #FF00FF;
}

/* visited link */
a:visited {
    color: #FF0000;
}

  /* unvisited link */
a:link {
    color:#00FF00; 
}



/* mouse over link */
a:hover {
    color:#0000FF;
}

Background Color

The background-color property specifies the background color for links:

Example

  a:link {
    background-color:#FF704D; 
}

a:visited {
    background-color:#FF704D; 
}

a:hover {
    background-color: #B2FF99;
}

a:active {
    background-color: #FFFF85;
}
