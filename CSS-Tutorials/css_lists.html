CSS Lists

CSS lists are simialr to the what you see in microsoft office bullets and square boxes before the content. Of course, that is provided by microsoft. However you can also do the same for your content on the web page.

CSS lists allows you do add different types of bullets,squareboxes for order and unorder lists. See the below image for more understanding

Basic Lists WithOut CSS

  <ul> 
  <li>Red</li>
  <li>Blue</li>
  <li>Green</li>
  <li>Yellow</li>
  </ul>

Basic Lists With CSS

  <!DOCTYPE html> 
<html>
 <head>
 <style> 
 ul { list-style-type: none; width:100px; }
  li { background-image: url(examples/images/list-icon.png);
   background-repeat: no-repeat; 
   background-position: 0px 5px;
    padding-left:15px; 
    }
 </style> 

 </head> 
 <body>
  <ul> 
     <li>Red</li>
     <li>Blue</li>
     <li>Green</li> 
     <li>Yellow</li>
  </ul>
 </body>
</html>

The HTML list properties allows you to style the content in three main different waysorder lists, unorder lists

You can further style these two lists according to your needs

CSS list items in diffetent ways

CSS lists items can be set using the list-style property

  
<!DOCTYPE html>
<html>
<head>
<style>
ul.x {
    list-style-type: circle;
}

ul.y {
    list-style-type: square;
}

ol.z {
    list-style-type: upper-roman;
}

ol.p {
    list-style-type: lower-alpha;
}
</style>
</head>
<body>

<h2>Example of unordered lists:</h2>
<ul class="x">
  <li>HTML</li>
  <li>CSS</li>
  <li>Javascript</li>
</ul>

<ul class="y">
  
  <li>HTML</li>
  <li>CSS</li>
  <li>Javascript</li>
</ul>

<h2>Example of ordered lists:</h2>
<ol class="z">
  
  <li>HTML</li>
  <li>CSS</li>
  <li>Javascript</li>
</ol>

<ol class="p">
  <li>HTML</li>
  <li>CSS</li>
  <li>Javascript</li>
</ol>

</body>
</html>

Setting an Image as a CSS list

Just as the circle, square boxes and others you can also set an image as the css list marker

<!DOCTYPE html>
<html>
<head>
<style>
ul {
    list-style-image: url('list-icon.png');
}
</style>
</head>
<body>

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Coca Cola</li>
</ul>

</body>
</html>

The Shorthand usage of CSS properties

  ul {
 list-style-type: square;
 list-style-position: outside;
 list-style-image: none;
}

You can also write the above css code using shorthand property

  ul {
    list-style: square outside none;
}

All CSS List Properties

Property Description Example
list-style Used to set all the properties for a list in a single declaration Try It Now
list-style-position It defines where to position the list marker, and it accepts one of two values: "inside" or "outside". Try It Now
list-style-image This property determines whether the list marker is set with an image, and accepts a value of "none" or a URL that points to the image: Try It Now
list-style-type The list-style-type property defines the type of list by setting the content of each marker, or bullet, on the list. Try It Now
