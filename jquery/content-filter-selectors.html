jQuery Content Filter Selectors




jQuery :contains() selector

jQuery contains selector selects all elements that contain the specified text.
Syntax 
  $( ":contains(text)" )

Example

The below example underlines the names which contains Gates as text in it

<!doctype html>
<html lang="en">
<head>
<meta charset="utf-8">
  <title>contains demo</title>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
  <div>Bill Gates</div>
  <div> Melinda Gates</div>

  <div>Jennifer Katharine Gates</div>
  <div> Rory John Gates</div>
  <div>Steve Jobs</div>
  <div>Mark Zuckerberg</div>
<script>
$( "div:contains('Gates')" ).css( "text-decoration", "underline" );
</script>
</body>
</html>

jQuery :empty Selector

jQuery empty selector , selects all elements that have no children (including text nodes).
Syntax 
  $( ":empty" )

Example

The below example finds all elements that are empty - they don't have child elements or text.

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>empty demo</title>
<style>
td {
text-align: center;
}
</style>
<script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
  <table border="1">
  <tr><td> #0</td><td></td></tr>
  <tr><td> #2</td><td></td></tr>
  <tr><td></td><td>#5</td></tr>
  </table>
<script>
$( "td:empty" )
.text( " Empty!" )
.css( "background", "red" );
</script>
</body>
</html>

It will select all the empty table columns and adds Empty text in that fileds also it changes the background to red

jQuery :has() Selector

jQuery has() selector, Selects elements which contain at least one element that matches the specified selector.
Syntax 
  $( ":has(selector)" )

Example

The below example adds blue border to the div which has a paragraph of text

<!doctype html>
<html lang="en">
<head>
<meta charset="utf-8">
<title>has demo</title>
<style>
.sample {
border: 3px inset blue;
}
</style>
<script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
  <div><p>Hello in a paragraph</p></div>
  <div>Hello again! (with no paragraph)</div><br/>
  <div><p>Another text with a paragraph</p></div><br/>
  <div>Hello! (with no paragraph)</div><br/>
<script>
$( "div:has(p)" ).addClass( "sample" );
</script>
</body>
</html>

:parent Selector

Select all elements that have at least one child node

Syntax
 jQuery( ":parent" )

This is the inverse of :empty

Example

Finds all tds with children, including text.

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>parent demo</title>
  <style>
  td {
    width: 40px;
    background: red;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<table border="1">
  <tr><td>Value 1</td><td></td></tr>
  <tr><td>Value 2</td><td></td></tr>
 <tr><td>Value 3</td><td></td></tr>
 <tr><td>Value 4</td><td></td></tr>
 <tr><td>Value 5</td><td></td></tr>
</table>
 
<script>
$( "td:parent" ).fadeTo( 700, 0.7 );
</script>
 
</body>
</html>
