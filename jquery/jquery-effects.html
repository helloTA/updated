jQuery Effects

The jQuery library provides several techniques for adding animation to a web page. These include simple, standard animations that are frequently used, and the ability to craft sophisticated custom effects.

jQuery Basic Effects

The following table lists all the jQuery methods of basic effects.

Method Description Example
show Shows the selected elements Try It
hide Hides the targeted elements Try It
toggle Toggles between the hide() and show() methods Try It

jQuery Custom Effects

The following table lists all the jQuery custom effects.

Method Description Example
animate Runs a custom animation on the selected elements Try It
clearQueue Removes all remaining queued functions from the selected elements Try It
delay Sets a delay for all queued functions on the selected elements Try It

jQuery Fading Effects

The following table lists all the jQuery methods of fading effects.

Method Description Example
fadeIn Fades in the selected elements Try It
fadeOut Fades out the selected elements Try It
fadeTo Fades in/out the selected elements to a given opacity Try It

jQuery Sliding Effects

The following table lists all the jQuery Sliding Effect Methods

Method Description Example
slideDown Slides-down (shows) the selected elements Try It
slideUp Slides-up (hides) the selected elements Try It
slideToggle Toggles between the slideUp() and slideDown() methods Try It
