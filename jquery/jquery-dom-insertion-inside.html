jQuery append() Method

The jQuery append() method in jQuery inserts specified content at the end of the selected elements which is specified by the parameter.

Syntax 
 .append(content )

The jQuery .append() method inserts the specified content as the last child of each element in the jQuery collection.

Example

<html lang="en">
<head>
  <meta charset="utf-8">
  <title>append demo here</title>
  <style>
  p {
    background: cyan;
font-size: 30px;
  }
span{font-weight:bold;}
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<span>Wann know more about jQuery!!!</span>
<p>Here is the append demo </p>
 
<script>
$( "p" ).append( $( "span" ) );
</script>
 
</body>
</html>

jQuery appendTo() Method

The append() method in jQuery inserts HTML elements at the end of the selected elements which is specified by the parameter.

Syntax for jQuery appendTo() method 
.appendTo(content)

The jQuery .append() and jQuery .appendTo() methods in jQuery performs the same task. The main difference is syntax i.e., placing content and target.

Example


<html lang="en">
<head>
  <meta charset="utf-8">
  <title>append demo here</title>
  <style>
  p {
    background: cyan;
font-size: 30px;
font-weight:bold;
  }
span{font-weight:normal}
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<span>Learn jQuery on Tutorials Academy</span>
<p>Here is the append demo </p>
 
<script>
$( "span" ).appendTo( $( "p" ) );
</script>
 
</body>
</html>

jQuery .html() Method

JQuery .html() method is used to get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element

Syntax for jQuery .html() Method

 .html()

In an HTML document, .html() can be used to get the contents of any element. If the selector expression matches more than one element, only the first match will have its HTML content returned.

Example for jquery .html() method

  $( "p" ).click(function() {
  var htmlString = $( this ).html();
  $( this ).text( htmlString );
});
