jQuery .class Selector

jQuery class selector finds elements with a specific class.

Selects all elements with the given class.

Syntax


 $(".test")

Example


 $(document).ready(function(){
    $( ".class-one" ).css( "border", "3px solid green" );
    $(".class-two").css("border", "5px solid blue");
    $(".class-three").css("border", "5px solid red");
});

If you run the above jQuery class example code, it will apply the green color for class-one and blue color to class-two and finally, red color to class-three. Run the below live example and see it in live.:
