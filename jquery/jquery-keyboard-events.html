jQuery keyboard Events




jQuery Keyboard Events

keyboard events are focused on the user keyboard while interacting with the web. It will make sure what needs to be done when the key on keyboard is pressed, what needs to be done when the downkeyboard on keyboard is pressed, what needs to be done when up arrow keyboard is pressed?? etc...

jQuery provides many keyboard methods. Let's see each of those jquery keyboard events in detail

jQuery keydown() Event

jQuery keydown() event is used to bind an event handler to an “keydown” javascript event or trigger that event on an element.

Syntax

keydown(handler )

jQuery keydown() method behaviour can also be performed by .on( "keydown", handler ) and trigger( "keydown" ). Basically keydown() event will be fired when user presses a key from keyboard.

Here is an example to use jquery keydown() method. In this example, the no.of times the event occurs will be shown as result when user try to presses the key on keyboard.

Example

  var xTriggered = 0;
$( "#target" ).keydown(function( event ) {
  if ( event.which == 13 ) {
   event.preventDefault();
  }
  xTriggered++;
  var msg = "Handler for .keydown() called " + xTriggered + " time(s).";
  $("div").html(msg);
});

jQuery keypress() Event

jQuery keypress() event is used to bind an event handler to a “keypress” javascript event or trigger that event on an element.

Syntax

keypress(handler )

jQuery keypress() method behaviour can also be performed by on( "keypress", handler ) and jQuery trigger( "keypress" ). Basically jQuery keydown() event will be fired when browser register keyboard event.

jQuery keypress()event is similar to the jquery keydown() event, except that modifier and non-printing keys such as Shift, Esc, and delete trigger key down events but not keypress events. Other differences between the two events may arise depending on platform and browser.

Here is an example on how to use jquery keypress() method.

Example

var xTriggered = 0;
$( "#target" ).keypress(function( event ) {
  if ( event.which == 13 ) {
   event.preventDefault();
  }
  xTriggered++;
  var msg = "Handler for .keypress() called " + xTriggered + " time(s).";
  $("div").html(msg);
});

In this example, the no.of times the event occurs will be shown as a result when user try to press the key on the keyboard.

jQuery keyup() Event

jQuery keyup() event is used to bind an event handler to a “key” javascript event or trigger that event on an element.

Syntax

keyup(handler )

jQuery keyup() method behaviour can also be performed by .on( "keyup", handler ) and jquery trigger( "keyup" ). The jquery key down() event will be fired when the user releases a key on the keyboard. This event only sent to the element that has the focus.

Example

var xTriggered = 0;
$( "#target" ).keyup(function( event ) {
  if ( event.which == 13 ) {
   event.preventDefault();
  }
  xTriggered++;
  var msg = "Handler for .keypress() called " + xTriggered + " time(s).";
  $("div").html(msg);
});

In this example, the no.of times the event occurs will be shown as a result when the user releases the key on the keyboard.
