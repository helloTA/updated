jQuery Basic Selectors

jQuery :even Selector

Selects even elements, zero-indexed.

Example

$( "tr:even" )

It selects all the even tr elements

<!doctype html>
<html lang="en">
  <head>
   <meta charset="utf-8">
  <title>even demo</title>
<style>
table {
background: #eee;
}
</style>
<script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
    <table border="1">
    <tr><td>Row 0 Even</td></tr>
    <tr><td>Row 1 Odd</td></tr>
    <tr><td>Row 2 Even </td></tr>
    <tr><td>Row 3 Odd</td></tr>
    <tr><td>Row 4 Even</td></tr>
</table>
<script>
$( "tr:even" ).css( "background-color", "red" );
</script>
</body>
</html>

jQuery :eq() Selector

Select the element at index n within the matched set.

Syntax

jQuery( ":eq(index)" )

Example

<!doctype html>
<html lang="en">
  <head>
   <meta charset="utf-8">
   <title>eq demo</title>
   <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
    <table border="1">
    <tr><td>One</td><td>Two</td><td>Three</td></tr>
    <tr><td>Four</td><td>Five</td><td>Six</td></tr>
    <tr><td>Seven</td><td>Eight</td><td>Nine</td></tr>
</table>
 <script>
 $( "td:eq( 2 )" ).css( "color", "green" );
 </script>
 </body>
</html>
