jQuery Basic Selectors




jQuery Basic Selectors

jQuery provides many selectors we can use on a webpage, in this chapter we will learn the basic selectors

jQuery even() Selector

Selects even elements, zero-indexed.

Example

$( "tr:even" )

It selects all the even tr elements

<!doctype html>
<html lang="en">
<head>
<meta charset="utf-8">
<title>even demo</title>
<style>
table {
background: #eee;
}
</style>
<script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
  <table border="1">
  <tr><td>Row 0 Even</td></tr>
  <tr><td>Row 1 Odd</td></tr>
  <tr><td>Row 2 Even </td></tr>
  <tr><td>Row 3 Odd</td></tr>
  <tr><td>Row 4 Even</td></tr>
  </table>
<script>
$( "tr:even" ).css( "background-color", "red" );
</script>
</body>
</html>

jQuery gt() Selector

Select all elements at an index greater than index within the matched set.

Syntax

  jQuery( ":gt(index)" )

Example


<!doctype html>
<html lang="en">
<head>
<meta charset="utf-8">
<title>gt demo</title>
<script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
  <table border="1">
  <tr><td>TD #0</td><td>TD #1</td><td>TD #2</td></tr>
  <tr><td>TD #3</td><td>TD #4</td><td>TD #5</td></tr>
  <tr><td>TD #6</td><td>TD #7</td><td>TD #8</td></tr>
  </table>
<br />
<script>
$( "td:gt(3)" ).css( "backgroundColor", "red" );

</script>
</body>
</html>

It will select all the td elements whose id is greater than 3. In other words it will select all the td elements from the index 4

jQuery not() Selector

Selects all elements that do not match the given selector.

Syntax

  jQuery( ":not(selector)" )

All selectors are accepted inside :not(), for example: :not(div a) and :not(div,a).

Example

<!DOCTYPE html>
<html>
<head>
 <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"></script>
<script>
$(document).ready(function(){
    $("p:not(.dont-select-me)").css("background-color", "green");
});
</script>
</head>
<body>

  <h1> My Homepage</h1>

  <p class="dont-select-me">Sunday.</p>
  <p>Monday.</p>
  <p>Tuesday.</p>
  <p class="dont-select-me">Wednsday</p>
  <p>Thursday</p>
  <p>Friday</p>
  <p>Saturday</p>
  </body>
</html>

It will select all the elements and apply the green color except for sunday anf webdensday.

jQuery :focus Selector

Selects element if it is currently focused.

Syntax

  jQuery( ":focus" )

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>focus demo</title>
<style>
.focused {
background: red;
}
</style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
  <div id="content">
  <input tabIndex="1">
  <input tabIndex="2">
  <select tabIndex="3">
  <option>select menu</option>
  </select>
  <div tabIndex="4">
  This is a div
  </div>
  <p>When you click inside the form it changes it background color to red
  </div>
<script>
$( "#content" ).delegate( "*", "focus blur", function() {
var elem = $( this );
setTimeout(function() {
elem.toggleClass( "focused", elem.is( ":focus" ) );
}, 0 );
});
  </script>
 </body>
</html>
