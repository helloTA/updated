jQuery first-child Selector

Selects all elements that are the first child of their parent.

Syntax

$( ":first-child" )

When there many no of elements in a page, jquery :first-child selector can be used to select the first element of each parent in the webpage

Example

    <!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>first-child demo</title>
  <style>
  span {
    color: #008;
  }
  span.sogreen {
    color: blue;
    font-weight: bolder;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<div>
  <span>Apple,</span>
  <span>Banana,</span>
  <span>Mango</span>
</div>
<div>
  <span>Orange</span>
  <span>Grape</span>
  <span>Carrot</span>
</div>
 
<script>
$( "div span:first-child" )
  .css( "color", "green" )
  .hover(function() {
    $( this ).addClass( "sogreen" );
  }, function() {
    $( this ).removeClass( "sogreen" );
  });
</script>
 
</body>
</html>

jQuery :last-child Selector

Selects all elements that are the last child of their parent.

When there are many no of elements in a webpage, and you want to select only the last element in each parent in such cases you can :last-child selector

Syntax

  $( ":last-child" )
Note: :last matches only a single element, :last-child can match more than one: one for each parent.

Example

Find the last span in each matched div and add some css hover effect.

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>last-child demo</title>
  <style>
  span.solast {
    text-decoration: line-through;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<div>
  <span>One,</span>
  <span>Two,</span>
  <span>Three,</span>
  <span>Four</span>
</div>
<div>
  <span>White,</span>
  <span>red,</span>
  <span>green,</span>
  <span>blue</span>
</div>
 
<script>
$( "div span:last-child" )
  .css({ color:"green", fontSize:"100%" })
  .hover(function() {
    $( this ).addClass( "solast" );
  }, function() {
    $( this ).removeClass( "solast" );
  });
</script>
 
</body>
</html>

Output

jQuery :first-of-type Selector

The :first-of-type selector matches elements that have no other element with both the same parent and the same element name coming before it in the document tree.

Syntax

  jQuery( ":first-of-type" )

When there are multiple elements with the same content or same value, jquery :first-of-type selector will select the first element of its type

Example

It will find the first span in each matched div and add a class to it.

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>first-of-type demo</title>
  <style>
  span.change {
    color: blue;
    font-size: 120%;
    font-style: bold;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<div>
  <span>Tom,</span>
  <span>Jerry,</span>
  <span>Tom,</span>
  <span>Cherry</span>
  <span>Tom,</span>
  <span>Tom,</span>
</div>
<div>
  <b>Dollar,</b>
  <span>Rupee,</span>
  <span>Euro,</span>
 <span>Rupee</span>
 <span>Rupee</span>
  
</div>
 
<script>
$( "span:first-of-type" ).addClass( "change" );
</script>
 
</body>
</html>

There are elements named Tom and 3 Elements named Rupee. But it only selected the first of its type.

jQuery :nth-child() Selector

Matches all elements that are the nth-child of their parent or that are the parent's even or odd children.

Syntax

$( ":nth-child(index/even/odd/equation)" )

index: The index of each child to match, starting with 1, the string even or odd, or an equation ( eg. :nth-child(even), :nth-child(4n) )

Example

The below example selects every second element in each parent of second child because in our case :nth-child(2)

<!doctype html>
<html lang="en">
<head>
<meta charset="utf-8">
<title>nth-child demo</title>
<style>
div {
float: left;
}
span {
color: blue;
}
p {
color: green;
}
</style>
<script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
  <body>
  <div>
  <ul>
  <li>One</li>
  <li>Two</li>
  <li>Three</li>
  </ul>
  </div>
  <div>
  <ul>
  <li>Numbers</li>
  <li>Names</li>
  <li>Values</li>
  </ul>
  </div>
  <div>
  <ul>
  <li>This is One</li>
  <li>This is Two</li>
  <li>This is Three</li>
  <li>This is Four</li>
  </ul>

  <p>It will selet every second element in each parent ul</p>
  </div>
<script>
$( "ul li:nth-child(2)" ).append( "<span> - 2nd!</span>" );
</script>
</body>
</html>
