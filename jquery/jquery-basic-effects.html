jQuery Show () Method

jQuery show() method is used to show the content wrapped inside HTML elements.

The jQuery show() method function call the css function .css( "display", "block") where display is a css property and block is the value to display element. This method doesn’t have animation.

Syntax

 .show()

Note: In css, If the element has the !important property in inline to the css styles then we need to override it by appending !important to the value of display attributes like css( "display", "block !important").

Consider the following example and which contains p tag

Example

  $("button").click(function() {
  $("p").show( "slow" );
});

jQuery hide() Method

jQuery hide() method simply hides the selected HTMLelements.

The hide() method works by calling .css( "display", "none") where display is a css property and none is the value to hide element. This method doesn’t have animation.

Syntax

.hide()

Note: In css, If the element has the !important property in inline css styles we need to override it by appending !important to the value of display attribute. .css( "display", "none !important")

Let us look at the following example to hide HTML element.

Example

for ( var i = 0; i < 5; i++ ) {
  $( "<div>" ).appendTo( document.body );
}
$( "div" ).click(function() {
  $( this ).hide( 3000, function() {
    $( this ).remove();
  });
});

jQuery toggle() Method

Before you understand about jquery toggle method try to understand these two jquery methods show and jquery hide

jQuery toggle() method does the both jquery show and jquery hide functions alternatively meaning if the element is initially displayed then it will be hidden, if the element is hidden then it will be shown. Internally toggle() method maintains the state of display property value.

Syntax

.toggle( [duration ] [, complete ] )

jQuery toggle() has a duration and functions parameters in order to make animation act upon the html element.

The following example illustrates how to use jquery toggle() method

Example

$( "button" ).click(function() {
  $( "p" ).toggle();
});

The above example will toggle the paragraph, like hiding and showing it
