jQuery Content Filter Selectors




:contains() Selector

When there are different types of content a single webpage, It select all elements that contain the specified text.

Synatx
  jQuery( ":contains(text)" )
Example
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>contains demo</title>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<div>Good Morning</div>
<div>Awesome Day</div>
<div>I am happy today</div>
<div>Good night</div>
 
<script>
$( "div:contains('Good')" ).css( "text-decoration", "underline" );
</script>
 
</body>
</html>

It selects all the elements in page which contains the text Good and it will add underline to that text

:empty Selector

Select all elements that have no children

Example
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>empty demo</title>
  <style>
  td {
    text-align: center;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<table border="1">
  <tr><td>TD #0</td><td></td></tr>
  <tr><td>TD #2</td><td></td></tr>
  <tr><td></td><td>TD#5</td></tr>
</table>
 
<script>
$( "td:empty" )
 
  .css( "background", "rgb(255,220,200)" );
</script>
 
</body>
</html>

Select all the td(s) which are empty and add some background

:parent Selector

Select all elements that have at least one child node

Syntax
 jQuery( ":parent" )

This is the inverse of :empty

Example

Finds all tds with children, including text.

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>parent demo</title>
  <style>
  td {
    width: 40px;
    background: red;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<table border="1">
  <tr><td>Value 1</td><td></td></tr>
  <tr><td>Value 2</td><td></td></tr>
 <tr><td>Value 3</td><td></td></tr>
 <tr><td>Value 4</td><td></td></tr>
 <tr><td>Value 5</td><td></td></tr>
</table>
 
<script>
$( "td:parent" ).fadeTo( 700, 0.7 );
</script>
 
</body>
</html>
