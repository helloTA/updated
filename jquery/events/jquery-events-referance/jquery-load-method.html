jQuery load() Method




jQuery load() Method

jQuery load() event is used to bind an event handler to the “load” javascript event.

Syntax

load(handler)

In general, the jquery load() event method will be called when all sub-elements have been completely loaded.

Here is an example to use jquery load() method. In the below example jquery load() method will be fired when image element gets loaded.

Example


<!DOCTYPE html>
<html>
<head>
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"></script>
<script>
$(document).ready(function(){
  $( "img" ).load(function() {
  if ( $( this ).height() > 100) {
    $( this ).addClass( "bigImg" );
alert('image is added');
  }
});
});
</script>
</head>
<body>

<div id="div1"><h2>Demo of document load() method in jQuery</h2></div>

</body>
</html>
