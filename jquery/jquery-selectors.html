jQuery Selectors

jQuery selectors are one of the powerful tools to manipulate HTML elements on a web page . It allows you to select and manipulate HTML element(s). jQuery selectors target a particular HTML element and change it's behavior. jQuery selectors must be matched with the existing HTML ids, class names, and type values. All selectors in the jQuery start with the dollar sign and parentheses: $().

What are jQuery Selcetors ?

A jQuery selector is a string which decides which HTML elements to target and manipulate it's data. jQuery selector string passed as an argument to $() or jQuery() selection function.


jQuery selectors are very easy to learn if you have a good knowledge on CSS selectors. Almost the syntax in CSS selectors and jQuery selectors is same.

jQuery follows below criteria to select HTML elements

  1. Element name (e.g. 'p', 'a', 'div' etc.)
  2. Element id
  3. Element CSS class
  4. Element attributes
  5. Element visibility
  6. Element order
  7. Form Fields
  8. Element parents or children
  9. Combinations of the above

jQuery Selector Example

The following example illustrates the jQuery Selector

$(document).ready(function(){
    $("#show").click(function(){
        $("p").show();
    });
});

This example selects the div element with the id #show, meaning the div element which has an id attribute with the value p.

jQuery Universal Selector

The jQuery universal selector is used to select all the elements on the webpage. Be sure when to use the jquery universal selector. If you misuse this universal selector it will change all the elements on the webpage.

Syntax

$("*")

Let's say if there 4 <h3> selsectors on a webpage and you want all selectors with one click.

HTML 
<body>
<h3> Main Heading</h3>
<h3>This is a heading 3.</h3>
<h3>This is a heading 3.</h3>
<h3>This is a heading 3.</h3>
</body>

You can select all <h3> elements on a page like this:

Example

jQuery 
 $(document).ready(function(){
    $("button").click(function(){
        $("*").hide();     // Select elements on the webpage
    });
});

jQuery id Selector

jQuery #id selector uses the id attribute of a HTML tag to find the specific element.

id should be unique within a page
Use the #id selector when you want to find a single or unique element.

Syntax

 $("#myid")

Let's say if there 4 h3 selectors on a webpage and you want all selectors with one click.

Example

HTML 
<body>
  <h2>Id Selector Example</h2>
  <p>This is a paragraph.</p>
  <p id="myid">I am going to hide if you click on the below button</p>
  <button>Click me</button>
</body>

In the above example code, we have a paragraph with an id called myid. Using the below jquery id selector we can simply target that myid element and apply the required styles we need.

jQuery 
$(document).ready(function(){
    $("button").click(function(){
        $("#myid").hide();
    });
});

When the clicks on button, the element with id=myid will be hidden:

jQuery class Selector

jQuery class selector is used to find all the elements which has same class name. Remember a class name what you initially give to yourHTML elements.

The jQuery class selector is can be called using . (DOT) operator followed by the class name

Syntax

 $(".example-class-name")

Example

HTML 
<div class="class-one">I am Class One"</div>
<div class="class-two">I am Class Two</div>
<span class="class-three">I am Class Three</span>

In the above HTML code each div element has a different class name as class-one, class-two, class-three. So what we can do is, we can simply target those three classes and apply the styles like below

jQuery 
 $(document).ready(function(){
    $( ".class-one" ).css( "border", "3px solid green" );
    $(".class-two").css("border", "5px solid blue");
    $(".class-three").css("border", "5px solid red");
});

If you run the above jQuery class example code, it will apply the green color to the class-one HTML div element, blue color to class-two div element and finally, red color to class-three span elemement

jQuery this selector

jQuery this selector is used to select the current HTML element and apply the required styles or manipulates the data.

jQuery $(this) selector enables you to select the current HTML element.

Syntax

$(selector).action()

Example

$(document).ready(function(){
   $("button").click(function () {
      $(this).next().removeAttr("disabled")
             .focus()
             .val("My next task is to go home");

jQuery Multiple Selector

jQuery multiple selector is used to target multiple html elements at a time and apply the required styles to HTML elements.

Syntax

$( "selector1, selector2, selectorN" )

Example

HTML 
<body>
  <div>div</div>
  <p class="classOne">p class="classOne"</p>
  <p class="notMyClass">p class="notMyClass"</p>
  <span>span</span>
</body>

In the above HTML code we have three different html elements those arediv two paragraph p elements and one span element

By using jquery multiple selector we can target all the four different html elements and apply the styles in a single as follows

jQuery 
$( "div, span, p.classOne" ).css( "border", "3px solid blue" );

The above jquery multiple selector will target div, p, span elements and apply css styles as css( "border", "3px solid blue" )
