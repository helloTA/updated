jQuery Selectors




jQuery Selectors

jQuery selectors are one of the most important key factors of the jQuery libraries. It allows you to select and manipulate HTML element(s). jQuery selection of particular HTML element can be done based on their id, classes, types, attributes, values of attributes and much more. All selectors in jQuery start with the dollar sign and parentheses: $().

jQuery Selector Overview

A jQuery selector is a string which decides which HTML elements to select. jQuery selector string passed as an argument to $() or jQuery() selection function.


jQuery selectors are very easy to learn if you have a good knowledge on CSS selectors. because of jQuery syntax is similar to CSS selectors.

jQuery follows following criteria to select HTML elements

  1. Element name (e.g. 'p', 'a', 'div' etc.)
  2. Element id
  3. Element CSS class
  4. Element attributes
  5. Element visibility
  6. Element order
  7. Form Fields
  8. Element parents or children
  9. Combinations of the above

jQuery Selector Example

The following example illustrates the jQuery Selector


$(document).ready(function(){
    $("button").click(function(){
        $("parenDiv").hide(); 
    });
});

This example selects the div element with the id parenDiv, meaning the div element which has an id attribute with the value theDiv.

jQuery All Selector

The jQuery element selector selects elements based on the element name.

You can select all <h3> elements on a page like this:

Syntax


$("*")

Example

When a user clicks on a button, all <h3> elements will be hidden:


 $(document).ready(function(){
    $("button").click(function(){
        $("*").hide();
    });
});

jQuery #id Selector

jQuery #id selector uses the id attribute of an HTML tag to find the specific element.

id should be unique within a page
Use the #id selector when you want to find a single, unique element.

Syntax


 $("#myid")

Example

When a user clicks on a button, the element with id="myid" will be hidden:


$(document).ready(function(){
    $("button").click(function(){
        $("#myid").hide();
    });
});

jQuery .class Selector

jQuery class selector finds elements with a specific class.

Selects all elements with the given class.

Syntax


 $(".test")

Example


 $(document).ready(function(){
    $( ".class-one" ).css( "border", "3px solid green" );
    $(".class-two").css("border", "5px solid blue");
    $(".class-three").css("border", "5px solid red");
});

jQuery this selector

Selects the current HTML element

$(this) enables you to select the current HTML element. Here, meaning of this keyword is to refer the current element.

Syntax

$(selector).action()

Example


$(document).ready(function(){
    $("button").click(function(){
        $(this).hide();
    });
});

jQuery Multiple Selector (“selector1, selector2, selectorN”)

Selects the combined results of all the specified selectors.


Syntax


$( "selector1, selector2, selectorN" )

Example

Finds the elements that match any of these three selectors.

$( "div, span, p.classOne" ).css( "border", "3px solid blue" );
