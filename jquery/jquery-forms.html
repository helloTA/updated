jQuery blur() Method

jQuery blur() method in jQuery is an event handler which is used to bind an event handler to “blur” JavaScript event or else trigger the event on a selected element in the DOM tree.

Syntax for jQuery blur() method 
 .blur(handler)

jQuery blur() method behaviour can also be performed by .on( "blur", handler ) and .trigger( "blur" )

The blur event is only suitable for form elements like <input> tag in Html. This event fires when a selected HTML element loses it’s focus

Here is the example to know how blur event functions

Example

  $( "#target" ).blur(function() {
  alert( ".blur() Handler is being called." );
});

Now let us see the example html code for the above jquery blur method

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>blur demo here</title>

  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>

<div id="other">
  Trigger the handler
</div>
<form>
  <input id="target" type="text" value="Field 1">
  <input type="text" value="Field 2">
</form> 
</body>
</html>

jQuery submit() method in jQuery is an event handler which is used to bind an event handler to “submit” JavaScript event or else trigger the event on a selected element in the DOM tree.

Syntax for jQuery submit() method 
 .submit(handler)

jQuery submit() method behaviour can also be performed by .on( "submit", handler ) and .trigger( "submit" ).

jQuery val() Method

jQuery submit() method in jQuery gets the current value of the first element in the selected element of DOM tree.

Syntax for jQuery val() method 
.val()

The jQuery val() method used to get the values of form elements like input, select and textarea. This method returns null value when no option selected of select html element.

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>val demo here</title>
  <style>
  p {
    color: black;
    margin: 4px;
  }
  b {
    color: orange;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<p></p>
 
<select id="single">
  <option>select option1</option>
  <option>select option2</option>
</select>
 
<select id="multiple" multiple="multiple">
  <option selected="selected">option1</option>
  <option>option2</option>
  <option selected="selected">option3</option>
</select>
 
<script>
function displayVals() {
  var singleValues = $( "#single" ).val();
  var multipleValues = $( "#multiple" ).val() || [];
  $( "p" ).html( "<b>Single selection :</b> " + singleValues +
    " <b>Multi selection :</b> " + multipleValues.join( ", " ) );
}
 
$( "select" ).change( displayVals );
displayVals();
</script>
 
</body>
</html>
