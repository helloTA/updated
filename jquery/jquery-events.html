jQuery Events




jQuery Events

jQuery makes it straightforward to set up event-driven responses on page elements. These events are often triggered by the end user's interaction with the page, such as when text is entered into a form element or the mouse pointer is moved. In some cases, such as the page load and unload events, the browser itself will trigger the event.

jQuery bind() Event

jQuery .bind() event in jQuery attach a handler to an event for the elements.

Syntax

 bind( eventType [, eventData ], handler )

The jQuery .bind() method is responsible for attaching an event handler directly to elements. Basically, Handlers are attached to the currently selected elements in the jQuery object, so those elements must exist when call .bind().

Example

$("p").bind("click", function( event ) {
  var str = "(" + event.pageX + ", " + event.pageY + ")";
  $("span").text( "Single Click done " + str );
});
$("p").bind("dblclick", function() {
  $("span").text( "Double Click done in " + this.nodeName );
});
$("p").bind( "mouseenter mouseleave", function( event ) {
  $(this).toggleClass("over");
});

jQuery error() Event

jQuery error() event is used to bind an event handler to an “error” in javascript event.

Syntax

error(handler )

Basically .error() method will be called when an element is not loaded properly such as images. So the error event sent to html element which are loaded by the browser and referenced by document.

Note: The event handler must be attached before the browser fires the error event, which is why the example sets the src attribute after attaching the handler. Also, the error event may not be correctly fired when the page is served locally; error relies on HTTP status codes and will generally not be triggered if the URL uses the file: protocol.

Here is an example to use jquery error() method. In this example we will come to see error event will be raised when an unwanted image source url given to image

Example

  $(document).ready(function(){
  $("img").error(function(){
      alert('error occured loading image');
  });
  $("button").click(function(){
      $("img").error();
  });
});

jQuery resize() Event

Basically jQuery resize() method will be called when a browser window size is changed such as images. So the error event sent to HTML element which are loaded by the browser and referenced by the document.

Syntax

    resize(handler )

Note: The event will be continuously sent to an element as long as the window resizing is in progress. jQuery resize() method doesn’t depend on how many times this handler is being called.

Here is an example to use jQuery resize() method. In this example, we will come to see the resizing value printed when browser window resized.

Example

$(document).ready(function(){
 $( window ).resize(function() {
  $( "body" ).prepend( "
 window size : " + $( window ).width() + "
" );
  });
});

jQuery scroll() Event

jQuery scroll() event is used to bind an event handler to a “scroll” javascript event or to trigger that event on an element.

Syntax

 scroll(handler )

The jQuery scroll() method will be called when scroll position is being changed. The jQuery scroll() event also raised when click up on the mouse, drag up on scrollbar, dragging inside the element, pressing the arrow keys or mouse scroll wheel.

Here is an example to use the jqueryscroll() method. In this example, we will come to see the fading effect which indicates that when a scroll position is being changed.
