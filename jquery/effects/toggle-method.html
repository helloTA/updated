jQuery Toggle Method




jQuery toggle() Method

jQuery toggle() method in jQuery does the both display and hide functions alternatively meaning if the element is initially displayed then it will be hidden, if the element is hidden then it will be shown. Internally toggle() method maintains the state of display property value.

Syntax

.toggle( [duration ] [, complete ] )

jQuery toggle() has a duration and functions parameters in order to make animation act upon the html element.

The following example illustrates how to use .toggle()

Example

$( "button" ).click(function() {
  $( "p" ).toggle();
});
