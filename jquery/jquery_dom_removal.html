jQuery DOM Removal




jQuery .detach() Method

jQuery .detach() method in jQuery removes selected elements from the DOM tree.

Syntax

 .detach()

The jquery .detach() method is the same as jquery .remove() but the only difference is detached method keeps the data of removed elements. If you want to reinsert removed element to DOM, it can be possible by .detach() method.

Here is an example to use .text() method

Example

$( "p" ).click(function() {
  $( this ).toggleClass( "off" );
});
var p;
$( "button" ).click(function() {
  if ( p ) {
    p.appendTo( "body" );
    p = null;
  } else {
    p = $( "p" ).detach();
  }
});

jQuery .empty() Method

jQuery .empty() method in jQuery removes all child nodes of matched elements from the DOM tree.

empty() method removes childelements and as well as text specific content in the matched element.

Syntax

    .empty()

Here is an example to use .empty() method

Example

$( "button" ).click(function() {
  $( "p" ).empty();
});

jQuery .remove() Method

jQuery .remove() method in jQuery remove matched elements from the DOM tree.

Syntax

  .remove()

jQuery remove() method also similar behaviour of .empty(). Here the difference is .remove() method additionally removes all bound events and jQuery data associated with elements.

Example

  $( "button" ).click(function() {
  $( "p" ).remove();
});

jQuery .replaceWith() Method

jQuery .replaceWith() method in jQuery replace each element with new content from matched elements in the DOM tree and finally returns the elements which was removed.

Syntax

  .replaceWith()

The jquery .replaceWith() method removes content from the DOM and inserts new content in its place with a single call.

Example

$( "button" ).click(function() {
  $( this ).replaceWith( "<div>" + $( this ).text() + "</div>" );
});

jQuery .replaceAll() Method

jQuery .replaceAll() method in jQuery replaces each target element from matched elements in the DOM tree.

Syntax

  .replaceAll()

The jquery .replaceAll() method is similar to .replaceWith(), but with the source and target reversed.

Example

    $( "<b>textReplaced   </b>" ).replaceAll( "p" );
