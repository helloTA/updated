jQuery Mouse hover() Event

jQuery mouse hover() event is used to bind more than one handlers to the matched elements and to be executed when the mouse pointer enters and leaves the elements.

Syntax

hover(handlerIn, handlerOut )

The jQuery hover() event method binds handlers for both mouseenter and mouseleave events. You can use it to simply apply behavior to an element during the time the mouse is within the element.

Example

  $( "li" ).hover(
  function() {
    $( this ).append( $( " *******" ) );
  }, function() {
    $( this ).find( "span:last" );
  }
);

In this example, blue stars(*****) will be displayed as a result when user mouse over on HTML li elements.

jQuery mouseup() Event

jQuery mouseup() event is used to bind an event handler to a “mouse up” javascript event.

Syntax

mouse up(handler )

Example

  $( "p" )
  .mouseup(function() {
    $( this ).append( "<br/><span style='color:green;'>Mouse up event fired.</span>" );
  })

In the above example when the user press the mouse up the jquery event gets triggered

jQuery mouseenter() Event

jQuery mousesenter() event is used to bind an event handler to be executed when the mouse enters an element.

Syntax

mouseenter(handler )

Example

  var i = 0;
$( "div.overout" )
  .mouseover(function() {
    $( "p:first", this ).text( "mouse over event occured" );
    $( "p:last", this ).text( ++i );
  })
  .mouseout(function() {
    $( "p:first", this ).text( "mouse out event occured" );
  });
 
var n = 0;
$( "div.enterleave" )
  .mouseenter(function() {
    $( "p:first", this ).text( "mouse enter event occured" );
    $( "p:last", this ).text( ++n );
  })
  .mouseleave(function() {
    $( "p:first", this ).text( "mouse leave event occured" );
  });

In the above example we can analyse when the mouse down and mouse up event occurs when press and releases on an HTML element.

jQuery mousemove() Event

jQuery mousemove() event in jquery is used to bind an event handler to an “mousemove” javascript event or trigger that event on an element.

Syntax

mousemove(handler )

jQuery mousemove() method behaviour can also be performed by .on( "mousemove", handler ) and .trigger( "mousemove" ). Basically mousemove() event will be fired when mouse pointer moves inside the element.

Example

  $( "div" ).mousemove(function( event ) {
  var pageCoords = "( " + event.pageX + ", " + event.pageY + " )";
  var clientCoords = "( " + event.clientX + ", " + event.clientY + " )";
  $( "span:first" ).text( "( event.pageX, event.pageY ) : " + pageCoords );
  $( "span:last" ).text( "( event.clientX, event.clientY ) : " + clientCoords );
});

In the above example, we can analyse when the mousedown and mouseup event occurs when mouse moves inside the html element.
