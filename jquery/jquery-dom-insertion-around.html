jQuery DOM Insertion Around





jQuery wrap() Method

jQuery wrap() method in jQuery wraps an HTML structure around each element in the group of matched element in DOM tree.

Syntax for jQuery wrap() Method 
  .wrap(wrappingElement)

The jQuery wrap() method takes a parameter as any type of string or object. HTML structure will be wrapped around each of the elements in DOM tree matched elements.

Consider the below HTML code

<p>Here is the jQuery</p>
<p>wrap demo</p>
<p>Learn it</p>

By using the jquery .wrap() method we wrap some html content around it like below

  $( "p" ).wrap( "<div></div>" );

Here is a complete example to use jQuery wrap() method

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>wrap demo here</title>
  <style>
  div {
    border: 5px solid red;
  }
  p {
    background: green;
    margin: 10px; color:white
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<p>Here is the jQuery</p>
<p>wrap demo</p>
<p>Learn it</p>
 
<script>
$( "p" ).wrap( "<div></div>" );
</script>
 
</body>
</html>

jQuery unwrap() Method

The jQuery unwrap() method removes the element's parent. This is effectively the inverse of the jQuery wrap() method. The matched elements (and their siblings, if any) replace their parents within the DOM structure.

Syntax for jQuery unwrap() method 
 .unwrap()

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>unwrap demo</title>
  <style>
  div {
    border: 8px solid red;
  }
  p {
    background: cyan;
    margin: 10px;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
  <button>wrap me/unwrap me</button>
<p>Hello</p>
<p>cruel</p>
<p>World</p>
<script>
var pTags = $( "p" );
$( "button" ).click(function() {
  if ( pTags.parent().is( "div" ) ) {
    pTags.unwrap();
  } else {
    pTags.wrap( "<div></div>" );
  }
});
</script>
 
</body>
</html>

jQuery wrapInner() Method

jQuery wrapInner() method in jQuery wraps an HTML structure around the content of each element in the group of matched element in DOM tree.

Syntax for jQuery wrapInner() Method 
 .wrapInner(wrappingElement)

Consider the following HTML:

<div class="container">
  <div class="inner">Tutorials</div>
  <div class="inner">Academy</div>
</div>

Using .wrapInner(), we can insert an HTML structure around the content of each inner <div> elements like so: 

$( ".inner" ).wrapInner( "<div class='new'</div>");

See the below example for more understanding

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>wrapInner demo</title>
  <style>
  p {
    background: #bbf;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<p>Welcome to</p>
<p>Tutorials Academy</p>

 
<script>
$( "p" ).wrapInner( "<b></b>" );
</script>
 
</body>
</html>
