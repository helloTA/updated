jQuery removeClass() Method

jQuery removeClass() method in jQuery removes the more than one specific class to the each element present in the set of matched elements.

synatx for jQuery removalClass(className) 
 .removeClass(className)

If a class name is mentioned as a parameter, then particular class will be removed from the group of matched elements. If no class names are specified in the parameter, all classes will be removed.

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>removeClass demo here</title>
  <style>
  p {
    margin: 8px;
    font-size: 22px;
    font-weight: bolder;
  }
  .blue {
    color: red;
  }
  .under {
    text-decoration: underline;
  }
  .highlight {
    background: green;color:white
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<p class="blue under">In jQuery</p>
<p class="blue under highlight">remove() method </p>
<p class="blue under">removes </p>
<p class="blue under">the classes</p>
 
<script>
$( "p:even" ).removeClass( "blue" );
</script>
 
</body>
</html>

jQuery toggleClass() Method

jQuery removeClass() method in jQuery add or removes the more than one specific class to the each element present in the set of matched elements.

Syntax for jQuery toggleClass() Method 
 .toggleClass(className)

This method accepts one or more class names as parameter. In which , if an element already has the class in matched element , then it removes the elements or if an element doesn’t have the class then it simply adds the element in the DOM.

jQuery clone() Method

The jQuery clone() method performs a deep copy of the selected elements in DOM tree, meaning that it copies the selected elements as well as all of their descendant elements and text nodes.

Syntax for jQuery clone() method 
  .clone(className)

This method accepts one or more class names as parameter. In which , if an element already has the class in matched element, then it removes the elements or if an element doesn’t have the class then it simply adds the element in the DOM.

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>clone demo</title>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<b>Here is the jQuery</b><p>Demo of clone() method</p>
 
<script>
$( "b" ).clone().prependTo( "p" );
</script>
 
</body>
</html>
