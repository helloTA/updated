jQuery Overview




jQuery Overview

Following steps are very useful to start with jQuery and which helps you to increase the confidence in building the web-based applications.

  1. Wait until the page is being ready.
  2. Select HTML elements that are defined to modify furtherly.
  3. Specify appropriate attributes with CSS styles to modify the behaviour.
  4. If needed, try adding the listener functions, e.g. click() or wait for a JQuery effect to finish etc.

Example


    // Step 1. Need to wait until page is being ready. this would be done by ready()
$(document).ready(function() {

    // Step 2. select the HTML element with id ‘parentDiv’ which actually defined in in html code.
    var parentDiv= $('#parentDiv);

    //Step 3. modify the HTML elements CSS attributes
    parentDiv.css("font-size", "13px");

    // Step 4. add a click-listener function as follows.
    parentDiv.click(function() {
        $('#parentDiv).hide(); // to hide the div element 
        });

}

First the above example by default JQuery's $(document).ready() function can be called when the page is being ready which takes a parameter as a function.

In Second step, jQuery select the HTML element by using the jQuery selection function called $() where it takes string as a parameter to identifying the particular HTML element like “parentDiv”.

In Third step, if we wanted to modify the styles of HTML element that can be achieved by simply taking the help of CSS styles. To achieve this, we have to use CSS() function with CSS attributes. As we see in the above example font-size is a CSS attribute and 13px is the value which sets the font-size of the parent div text.

In Final step, we can add jquery events for the HTML elements like click(), dblclick() and mouseleave() etc. From example jQuery adds the click listener to HTML element “parentDiv” which can be done by using the click() function. Click listener will be functioned when click on HTML “parentDiv” element.
