jQuery Overview




jQuery Overview

In this overview chapter, lets look at how jQuery actually works with html elements and implement the jquery functions or methods on it.

Following steps are very useful to start with jQuery and which helps you to understand how jQuery internally works with HTML elements.

  1. Wait until the page is being ready.

  2. Select HTML elements that are defined to modify furtherly.

  3. Specify appropriate attributes with CSS styles to modify the behaviour.

  4. Try adding the listener functions, e.g. click() or wait for a JQuery effect to finish etc.

Example


//Step 1. Make sure you wait till the page is ready. This will be done by ready().

// Step 2. Select the HTML element with id ‘#hide’ which is actually defined in in html code.
// Step 3. Modify the HTML elements CSS attributes
// Step 4. Add a click-listener function . 

$(document).ready(function(){
    $("#hide").click(function(){
        $("p").hide();
    });
    $("#show").click(function(){
        $("p").show();
    });
});
Try It Now

In the above example by default jQuery's $(document).ready() function will be called when the page is being ready which takes a parameter as a function.

In the second step, jQuery will select HTML element by using the jQuery selection function called $() where it takes string as a parameter to identify the particular HTML element like #hide and #show.

In eh third step, if we want to modify the styles of HTML element that can be done by simply taking the help of CSS styles. To do this, we have to use CSS() function with CSS attributes.

In the final step, we added jquery events for the HTML elements like click(). Then jQuery adds the click listener to HTML element #hide and #show which is done using the click() function. Click listener will be functioned when click on HTML #hide and #show element.
