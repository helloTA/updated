jQuery addClass() Method

jQuery addClass() method in jQuery adds the more than one specific class to the each element present in the set of matched elements.

This method doesnâ€™t remove the existing class attributes but extends or adds the more that one class attributes to existing class attribute. Note: If you want to mention more than one class attribute, you have to separate each class name by comma.


Syntax for jQuery addClass() method 
 $addClass( className )

The following example illustrates the usage of jQuery addClass() method. In this example, it is going to add two more classes attributes to the existing <p> tag css attributes by calling $( "p:last" ).addClass( "selected highlight" ) where selected and highlight are the css classes defined under the <style> tag.

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>addClass demo</title>
  <style>
  p {
    margin: 10px;
    font-size: 20px;
  }
  .selected {
    color: white;
  }
  .highlight {
    background: red;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<p>Hi</p>
<p>This is jQuery</p>
<p>Learning platform</p>
 
<script>
$( "p:last" ).addClass( "selected highlight" );
</script>
 
</body>
</html>

jQuery css () Method

The CSS() method gets one or more style properties for the selected elements.

The jQuery css() method simplest way to get the computed style property from the selected element. jQuery takes care of CSS and DOM formatting of multiple word properties that are defined in jQuery css() method.


Syntax for jQuery css() method

 .css( propertyName )

For Example, Both .css( "background-color" ) and .css( "backgroundColor" ) can be understood by jQuery and returns the correct value. Meaning; these mixed case has a special meaning but incase if we provide .css( "WiDtH" ) and .css( "width" ), jQuery ignores and return correct value.

The following example gives the clarity of how to use .css() method.

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>css demo</title>
  <style>
  div {
    width: 100px;
    height: 100px;
    margin: 12px;
    float: left;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<span id="result"> </span>
<div style="background-color:yellow;"></div>
<div style="background-color:rgb(25,09,90);"></div>
<div style="background-color:#042490;"></div>
<div style="background-color:#f00;"></div>
 
<script>
$( "div" ).click(function() {
  var color = $( this ).css( "background-color" );
  $( "#result" ).html( "That div is <span style='color:" +
    color + ";'>" + color + "</span>." );
});
</script>
 
</body>
</html>

jQuery hasClass() Method

The hasClass() method in jQuery responsible for checking whether the selected element matches the specified class name or not. If matched hasClass() method returns true otherwise It returns false.

Syntax for jQuery hasClass() method 
 .hasClass(className)

Consider the example to check whether selected element matches the by class name

Example

jQuery height() Method

The height() method in jQuery gets the computed height of the matched element or Set the height of matched element.

Syntax for jQuery .height() method 
 .height()

The difference between .css( "height" ) and jQuery .height() is that the latter returns a unit-less pixel value (for example, 400) while the former returns a value with units intact (for example, 400px). The jQuery .height() method is recommended when an element's height needs to be used in a mathematical calculation.

Here is an example to use the jQuery .height() method.

Example 

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>height demo</title>
  <style>
  body {
    background: gray;
  }
  button {
    font-size: 18px;
    margin: 8px;
  }
  p {
    width: 190px;
    border: 2px blue solid;
  }
  div {
    color: white;
    font-weight: bold;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<button id="getp">Paragraph Height</button>
<button id="getd">Document Height</button>
<button id="getw">Window Height</button>
 
<div> </div>
<p>
  example to test height() method
</p>
 
<script>
function showHeight( element, height ) {
  $( "div" ).text(element + " height : " + height + "px." );
}
$( "#getp" ).click(function() {
  showHeight( "paragraph", $( "p" ).height() );
});
$( "#getd" ).click(function() {
  showHeight( "document", $( document ).height() );
});
$( "#getw" ).click(function() {
  showHeight( "window", $( window ).height() );
});
</script>
 
</body>
</html>

jQuery offset() Method

jQuery offset() method in jQuery allows you to get or set the current coordinates of the first element from the matched elements in DOM tree.

Syntax for jQuery offset() method 
 .offset()

Basically, this method helps us to retrieve the current position of an element which is relative to the DOM.

Here is the example to know how to get coordinates of the element by using offset() method.

Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>offset demo</title>
  <style>
  p {
    margin-left: 30px;
color: green;
font-size:30px;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<p>jQuery offset method</p><p>Learn it</p>
 
<script>
var p = $( "p:last" );
var offset = p.offset();
p.html( "left: " + offset.left + ", top: " + offset.top );
</script>
 
</body>
</html>
