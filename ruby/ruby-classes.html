Ruby Classes and Ruby Objects



Ruby is an object-oriented programming language, which means it manipulates programming constructs called objects. Sometimes we refer class as blueprint of an object.

A class is made up of a collection of variables representing internal state and methods providing behaviours that operate on that state.

Classes are defined in Ruby using the class keyword followed by a name.The name of the ruby class must begin with a capital letter.

In the above image you can say Car is a class.

if you create an object for the car like below.

BMW = Car.new

then BMW is the instance of Car class.

A basic class consists only of the class keyword and the name of the class.

Ruby Class Syntax


  class Myclass
   # code goes here
  end

Ruby Class Basic Example


class Company
   # Every class must be initialized before it is used
  def initialize(company, founder)
    @company_name = company
    @company_founder = founder
  end

# In object oriented programming class can have many methods. 
# Here we've one method called description	

  def description
    puts "The company is #{@company_name} and it was founded by #{@company_founder}"
  end
end

# In object oriented programming, we can create any number of
# objects or instances for the class. Here we've 3 objects

obj1 = Company.new("Microsoft", "Bill Gates")
obj2 = Company.new("Facebook", "Mark Zuckerberg")
obj3 = Company.new("Apple", "Steve Jobs")

obj1.description
obj2.description
obj3.description



 OutPut

The company is Microsoft and it was founded by Bill Gates
The company is Facebook and it was founded by Mark Zuckerberg
The company is Apple and it was founded by Steve Jobs

Every ruby class can have any no of variables and methods. Lets say, if we've a class called Car. It will contain properties. Please see the image below for more understanding.

I hope you understand about the ruby classes by now. Lets take a look at ruby objects.

Ruby Objects

What is an Object? An object is nothing but the instance of a class. In object oriented programming objects are called the blueprints of a class. Lets say you’ve a car class. Here Car is a class. Lets say you’ve BMW, Ferrari cars. Now these BMW is an object of Car class, and ferrari is another object of the same Car class.

Object plays an important role in Object oriented programming language. In we Ruby programming language we can create objects in our programs to manipulate data using new keyword. In Ruby new keyword is used to create a new object.

Lets look at the syntax of how we can create an object in ruby. 



 Ruby Object Syntax

  bmw= Car.new
  # If you want to pass the data then you can pass some parameters.
  bmw = Car.new(“Red color”, “BMW”, $100000)

Ruby Objects Example


class Car
  @@vehicle_type = "4 wheeler"

  def initialize(color, brand, horsepower)
   @color = color
   @brand = brand
   @horsepower = horsepower
  end

  # Our car class can have any no of methods.
  # In this Car example we've 4 methods.
  # A Car can go fast, go slow, turn_back, turn_left

  def car_details
   puts "I've a #{@color} color #{@brand} car and it is #{@horsepower} fast. "
  end

  def fast
   puts "Drive the car fast...."
  end

  def slow
   puts "Drive the car slowly.."
  end

  def move_back
   puts "Turn Back"
  end

  def move_left
      puts "Turn left"
  end
end

# Creating an Object for Car Class

bmw = Car.new("Red", "BMW", "100 KM/HR")

# Creating another object for Car Class
ferrari = Car.new("Blue", "Ferrari", "250 KM/HR" )

# calling our class methods with the class instance(object)

bmw.car_details
ferrari.car_details

# calling other class methods

bmw.fast
ferrari.slow

bmw.move_left
bmw.move_back



 Output

I've a Red color BMW car and it is 100 KM/HR fast.
I've a Blue color Ferrari car and it is 250 KM/HR fast.
Drive the car fast....
Drive the car slowly..
Turn left
Turn Back
