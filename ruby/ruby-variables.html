Ruby variables



Variables are used to store data in the memory. It contains some data in our program.

Syntax:

  


var_name = “value”

Each variable has a name and value.
Just like the above sentence a car needs some  parking space to keep it in real world.
In computers and programs we need place to keep our data and make it as reference to call it or see it or use it later.
I am hoping you could understand what is a variable means by now.

Almost every programming language has variables but all are not same. In ruby, we’ve 4 different variables for 4 different purposes. Lets see what they are and why they are.
Local Variables:
A variable whose name begins with a lowercase letter (a-z) or underscore (_) is a local variable.
IMAGE - ruby-local-variables.jpg
Key:  A local variable is only accessible from within the block of its initialization
Example: 

  loop {

  var_two =  100      # only accessible within this  block

  puts "The  value of local variable  is  #{var_two}"

  break  

  }
output: 

  The value of local variable is 100

In the above example var_two is a local variable which means it can be accessed only within the block. i.e in between curly braces “{ }”
Example 2:
Lets try to access the local variable from outside of the block. See the ruby interpreter complains about that.

  loop {

  loop {

  var_two = 100

  }

  puts “The value of local variable is #{var}”    # Accessing from outside
output: 

  Error:  undefined local variable or method `var_two'

  When you access a local variable from outside of the loop or block it  gives 

  “ undefined local variable or  method `var_two' “
Try an Example:  repl.it/languages/Ruby

Global Variables

Global variables can be accessed from anywhere within the program during runtime.
A variable whose name begins with '$' has a global scope
IMAGE - ruby-global-variables.jpg

Example:


  $var_one  = 10            # can be accessed from anywhere(Global Variable )

  loop {

  var_two =  100      # only accessible within this  block

  puts "The  value of local variable  is  #{var_two}"

  break            

  }

  puts “The value of global variable is #{var_one}”
output: 

  The value of local variable is 100

  The value of global variable is 10
Intermediate Example:

  def method_one

  $number =  100  # Global variable

  puts  "the number is #{$number}"  #  it should print 100

  end

  method_one
Output: 

  the number is  100
  


def method_one

  $number =  100            # assigning number = 100

  puts "the  number is #{$number}"     # it will  print as number = 100

  end

  method_one

  def method_two

  $number =  200     # changing the value of number to  200

  puts "the  number is #{$number}"   # it will  print as number = 100 because it has been changed

  end

  method_two
Output:    

the number  is 100

the number  is 200

Ruby Instance Variables

A variable whose name begins with '@' is an instance variable of self.
An instance variable belongs to the object itself. Uninitialized instance variables have a value of nil. Instance variables do not need to be declared
Every instance of a class has a different set of instance variables

Ruby Class Variables

A class variable is shared by all instances of a class and begins with '@@'

Example: 


  # In this ruby program we'll focus on ruby class, ruby  object 

  # and how we can change the class variables in ou program.

  class School

  @@students_count =  10   # these 3 class variables can be  accessible

  @@teachers_count =  4    # in the class. If you change

  @holidays =  15          # class variables in any  method it will be final.


  def  initialize(student_name, student_rank, student_class)

  # these  parameters are used to catch the values we pass 

  # during the  object creation. Refer to Object creation below

  @s_name =  student_name   # Instance variables

  @s_rank =  student_rank

  @s_class =  student_class

  end


  def student_info

  puts  "Displaying Student Details"

  puts  "Student name is #@s_name"

  puts  "Student rank is #@s_rank"

  puts  "Student class is #@s_class"


    # Lets also  display the current details

  puts "The  Old students count is #@@students_count"

  puts "The  Old teachers count is #@@teachers_count"

  end


  ## Lets increase  the student count by 4 and teachers count by 1

  def  total_students_count()

  @@students_count  += 4          # Accessing the class  variable @@students_count

  puts "The  new students count #@@students_count"

  end


  ## Lets  also increase the teachers count by 1.

  def  total_teachers_count()

  @@teachers_count  += 1       # Accessing the class variable  @@teachers_count

  puts "The  new teachers count is   #@@teachers_count"

  end

  end


# Now lets create object for the class and pass some data


student_one = School.new("Mark",  "1st", "5th")

  student_two = School.new("Steve",  "2nd", "6th")


# Lets call the methods we've defined.


student_one.student_info

  

  student_one.total_students_count

  student_one.total_teachers_count
