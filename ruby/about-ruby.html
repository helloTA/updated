About Ruby



Ruby is a dynamic, object-oriented, general -purpose programming language. It was designed and developed in the mid-1990s by Yukihiro Matsumoto in japan. Ruby supports multiple programming paradigms. Ruby was originally influenced by Perl, SmallTalk Ada,

Ruby has a dynamic type system and automatic memory management.

The name "Ruby" originated during an online chat session between Matsumoto and Keiju Ishitsuka on February 24, 1993

The first public release of Ruby 0.95 was announced on Japanese domestic newsgroups on December 21, 1995

Ruby Runs on a variety of platforms, such as Windows, Mac OS, and the various versions of UNIX/Linux. This tutorial is for beginners to help understand the basic to advanced concepts in Ruby Scripting languages.

The Author of Ruby Language. Yukihiro Matsumoto

Ruby is designed for programmer productivity and fun, following the principles of good user interface design

Ruby is said to follow the principle of least astonishment (POLA), meaning that the language should behave in such a way as to minimize confusion for experienced users

Ruby code can programmatically modify, at runtime, aspects of its own structure that would be fixed in more rigid languages.

The syntax of Ruby is broadly similar to that of Perl and Python

One of the differences of Ruby compared to Python and Perl is that Ruby keeps all of its instance variables completely private to the class and only exposes them through accessor methods

Ruby is object-oriented: every value is an object, including classes and instances of types that many other languages designate as primitives

Why use Ruby

  • Ruby is easy to learn
  • Ruby has many great frameworks
  • Ruby is flexible
