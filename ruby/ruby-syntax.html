Ruby is the programmer's best friend.  That being said, ruby is very easy. The language ruby can be understood by non-programmers as well.  In this section, I’ll explain the basic syntax of ruby. Let’s get started with ruby operators.

Ruby Assignment:
Ruby assignment is done using the equal operator “=”. This is both for variables and objects.


name = “tutorialsacademy”                     #assigning string to a  variable

year = 2015                                              #  assigning number to a variable

Ruby multiple assignment:

ruby multiple assignment allows us to assign more than one variable in the same line with (,) comma separation.

  
  num1, num2, num3 = 10, 11, 12

  puts num1        #prints 10

  puts num2       #prints 11

  puts num3       #print 12
