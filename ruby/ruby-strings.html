Ruby Strings



A String object holds and manipulates an arbitrary sequence of bytes, typically representing characters.
One way to create a String is to use single or double quotes inside a Ruby program to create what is called a string literal.


puts 'Hello world'

puts "Hello world"
So what difference is there between single quotes and double quotes in Ruby? In the above code, there's no difference. However, consider the following code:

puts "Bob’s  coffee machine"

puts 'Bobs\'s coffee machine'

Because "Bob's" contains an apostrophe, which is the same character as the single quote, in the second line we need to use a backslash to escape the apostrophe so that Ruby understands that the apostrophe is in the string literal instead of marking the end of the string literal. The backslash followed by the single quote is called anescape sequence.

Ruby Strings Single quotes

Single quotes only support two escape sequences.

  1. \' – single quote
  2. \\ – single backslash

Except for these two escape sequences, everything else between single quotes is treated literally.
Ruby String Double Quotes
Double quotes allow for many more escape sequences than single quotes. They also allow you to embed variables or Ruby code inside of a string literal – this is commonly referred to as interpolation.


  puts "Enter  name"

  name = gets.chomp

  puts "Your name is #{name}"

Escape sequences[edit]

Below are some of the more common escape sequences that can appear inside of double quotes.

  1. \" – double quote
  2. \\ – single backslash
  3. \a – bell/alert
  4. \b – backspace
  5. \r – carriage return
  6. \n – newline
  7. \s – space
  8. \t – tab


  puts "Hello\rStart over  world"

  puts "1. Hello\n2. World"

The result:

  Hello            world

  Goodbye world

  Start over world

  1. Hello

  2. World

Ruby's print function only prints out a new line if you specify one. For example, try out the following code. We include a newline at the end of print's argument list so that the shell prompt appears on a new line, after the text.

  print "Say", "hello", "to", "the", "world", "\n"
The result:

  Sayhellototheworld
Ruby provides so many methods we can apply with strings. Lets look at those.
capitalize    ruby capitalize method will capitalize the first letter in any string.

  str =  “apple”

  str.capitalize  # => Apple

  my_string = “123abc”

  my_string.capitalize # => 123ABC

count ruby count method will count the no of letters in a string


  str = “Good Morning”

  str.count(“o”)  # => 3

downcase ruby downcase method will change the upcase letters to downcase


  
 fruit = “BaNanA”

  fruit.downcase   # => banana        

empty? ruby empty method will check if the string is empty or not. If empty , it returns true


  str = “hello”

  str.empty?   # => false

  my_str = “”

  my_str.empty?   # => true

concat ruby concat method will concat the two strings.


  str = “hello ”

  str.concat(“  world”)

  str  # => helloworld

include? ruby include? method will checks if the given character or string is present or not.


  my_str  = “Morning” 

  my_str.include?(“n”)  # => true

  my_str.include?(“p”)  # => false

replace ruby replace method will replace the given string or character with old character of string


  fruit = “banana”

  fruit  # => banana

  fruit.replace(“orange”)

  fruit # => orange

insert ruby insert method will insert any character at a given index


  str = “bad”   # => bad

  str.insert(“0”, “m”)

  str # => mad

prepend ruby prepend method will add any string or character before other string or character


  my_life =  “Night”

  my_life.prepend(“Good”)

  my_life # =>  Good Night

upcase ruby upcase method will change the downcase string to upcase


  a = “hello”

  a.upcase   # => HELLO
