Installing Ruby on Windows

Installing Ruby on Windows is pretty simple. Follow the below instructions and you'll be ready to execute ruby programs on your windows machine.

Step 1

Head over the Ruby Official Website Ruby Website and click "Download Ruby" on the right.

Step 2

scroll down to the 'Ruby on Windows' section, you'll see that the easiest way to do it is by using the RubyInstaller. Choose the latest version of Ruby, which is Ruby 1.9.2p0. Download it and install it, just like any other Windows program.

step 3

Once you've got it installed, it's pretty simple to make sure everything worked. Just open a command prompt and type this:


 $ ruby --version
ruby 1.9.2p0 (2010-08-18) [i386-mingww32]

You will see the above message if you installed ruby successfully.

It's that easy!
